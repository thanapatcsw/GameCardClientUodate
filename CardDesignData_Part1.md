# ข้อมูลการ์ดเกม Startup City (ฉบับร่าง 50% - เน้น CPU, RAM, Storage)

เอกสารนี้รวบรวมข้อมูลการ์ดจำนวน 54 ใบ (จากทั้งหมด 90 ใบ) โดยอิงตามแนวคิดวิทยาการคอมพิวเตอร์ (Computer Science) และวิศวกรรมซอฟต์แวร์ ซึ่งสอดคล้องกับ TOR ของระบบการศึกษาและโครงสร้างเกมระดับ Tier 1 ถึง Tier 3 เพื่อให้คะแนนเหมาะสมกับเงื่อนไข **ชนะที่ 20 คะแนน (แต้ม)** จึงมีการปรับสมดุลคะแนนดังนี้:
- **0 ดาว**: 0 แต้ม (เน้นสะสมทรัพยากร/Engine Building)
- **1 ดาว**: 1 แต้ม (ทรัพยากรพื้นฐานที่มีมูลค่าแต้ม)
- **2 ดาว**: 2 แต้ม (ตัวทำคะแนนช่วงกลางเกม)
- **3 ดาว**: 4 แต้ม (ตัวเปลี่ยนเกม)
- **4 ดาว**: 5 แต้ม (ตัวเร่งเครื่องเข้าสู้ท้ายเกม)
- **5 ดาว**: 7 แต้ม (เทคโนโลยีระดับพลิกโลก)

## 🟥 การ์ด CPU สีแดง (หมวดการประมวลผลและการคำนวณตรรกะ)
*เน้นไปที่ลอจิก, โครงสร้างโปรเซสเซอร์, และเทคนิคการประมวลผลขั้นสูง*

### Tier 1 (8 ใบ)
- **0 ดาว (6 ใบ):**
  1. **Transistor (ทรานซิสเตอร์):** [0 แต้ม] อุปกรณ์เซมิคอนดักเตอร์พื้นฐาน สวิตช์เปิดปิดกระแสไฟฟ้า นำมาประกอบกันเป็นลอจิกเกตเพื่อเริ่มการคำนวณ
  2. **Logic Gate (ลอจิกเกต):** [0 แต้ม] วงจรที่รับสัญญาณอินพุตและให้เอาต์พุตตามตรรกะบูลีน (เช่น AND, OR, NOT) อิฐก้อนแรกของคอมพิวเตอร์
  3. **ALU (Arithmetic Logic Unit):** [0 แต้ม] วงจรดิจิทัลที่ทำหน้าที่คำนวณทางคณิตศาสตร์และประมวลผลตรรกศาสตร์
  4. **Clock Signal (สัญญาณนาฬิกา):** [0 แต้ม] ตัวกำหนดจังหวะการทำงานของโปรเซสเซอร์ เพื่อให้ทุกส่วนสอดคล้องกัน
  5. **Control Unit (หน่วยควบคุม):** [0 แต้ม] แปลคำสั่งและควบคุมการไหลของข้อมูลใน CPU ดั่งวาทยกรในวงดนตรี
  6. **Syntax & Logic (ไวยากรณ์เชิงตรรกะ):** [0 แต้ม] โครงสร้างไวยากรณ์พื้นฐานในการสั่งให้คอมพิวเตอร์ประมวลผล (if-else, loops)
- **1 ดาว (2 ใบ):**
  7. **Microprocessor (ไมโครโปรเซสเซอร์):** [1 แต้ม] วงจรรวมที่รวมฟังก์ชันหลักทั้งหมดของ CPU สมัยแรกไว้ในชิปเดียว
  8. **Pipelining (ระบบไพป์ไลน์):** [1 แต้ม] เทคนิคการซ้อนทับการประมวลผลคำสั่งแต่ละขยัก เพื่อเพิ่ม Throughput 

### Tier 2 (6 ใบ)
- **1 ดาว (2 ใบ):**
  9. **Task Scheduler (ตัวจัดคิวงาน):** [1 แต้ม] โค้ดในระบบปฏิบัติการที่จัดการแบ่งเวลาของ CPU ให้กับแอปพลิเคชันต่างๆ อย่างเป็นธรรม
  10. **Interrupt Handler (รหัสจัดการขัดจังหวะ):** [1 แต้ม] กลไกตอบสนองด่วนเมื่อฮาร์ดแวร์หรือระบบเกิดการร้องขอขัดจังหวะ
- **2 ดาว (2 ใบ):**
  11. **Multi-Core Processor (ซีพียูหลายแกน):** [2 แต้ม] การบรรจุ CPU หลายตัวไว้ในชิปวงจรเดียว ช่วยให้ทำงานจริงแบบขนานได้
  12. **Multithreading (มัลติเธรดดิง):** [2 แต้ม] เทคนิคทางซอฟต์แวร์ที่แบ่งกระบวนการทำงานเดียวออกเป็นหลายเธรดเพื่อให้รันพร้อมกันได้
- **3 ดาว (2 ใบ):**
  13. **Hyper-Threading (ไฮเปอร์เธรด):** [4 แต้ม] เทคโนโลยีที่ทำให้คอร์ประสิทธิภาพสูง 1 คอร์ทางกายภาพ ทำงานจำลองเสมือนมี 2 คอร์ 
  14. **GPU (Graphics Processing Unit):** [4 แต้ม] หน่วยประมวลผลคู่ขนานที่เชี่ยวชาญการคำนวณเมทริกซ์และภาพกราฟิก

### Tier 3 (4 ใบ)
- **3 ดาว (1 ใบ):**
  15. **Server Farm CPU (คอร์ประมวลผลแม่ข่าย):** [4 แต้ม] ชิปตระกูลสถาปัตยกรรมระดับองค์กรที่ทำงานอึดและรองรับแบนด์วิดท์หนักๆ
- **4 ดาว (2 ใบ):**
  16. **Neural Processing Unit (ชิปโครงข่ายประสาทแบบ NPU):** [5 แต้ม] ชิปที่ถูกวางโครงสร้างไว้สำหรับประมวลผลโมเดล AI (Deep Learning) อย่างเจาะจง
  17. **Quantum Processor / QPU (ซีพียูควอนตัม):** [5 แต้ม] ชิปประมวลผลตามกลศาสตร์ควอนตัม (Qubits) สำหรับแก้ปัญหาซับซ้อนระดับโลก
- **5 ดาว (1 ใบ):**
  18. **Supercomputer Cluster (คลัสเตอร์ซูเปอร์คอมพิวเตอร์):** [7 แต้ม] นวัตกรรมรวมโปรเซสเซอร์นับล้านเข้าด้วยกันเพื่อวิเคราะห์และคำนวณปัญหาใหญ่สุดแห่งยุค

## 🟦 การ์ด RAM สีฟ้า (หมวดหน่วยความจำ โครงสร้างข้อมูล และการจัดการสถานะ)
*เน้นไปที่การรับส่งข้อมูลไว, อัลกอริทึม, และการจัดการพื้นที่ชั่วคราว*

### Tier 1 (8 ใบ)
- **0 ดาว (6 ใบ):**
  1. **Bit & Byte (บิตและไบต์):** [0 แต้ม] หน่วยข้อมูลพื้นฐานที่สุดของข้อมูลดิจิทัลที่บ่งบอกสถานะ (0 หรือ 1)
  2. **Variable (ตัวแปร):** [0 แต้ม] พื้นที่ในหน่วยความจำ RAM ที่สร้างผ่านโค้ดเพื่อเก็บค่าข้อมูลต่างๆ
  3. **Pointer (พอยน์เตอร์):** [0 แต้ม] ตัวแปรที่เก็บค่าที่อยู่ (Address) ของข้อมูลในหน่วยความจำโดยตรง
  4. **Registers (เรจิสเตอร์):** [0 แต้ม] หน่วยความจำขนาดเล็กที่สุดแต่อยู่ใกล้ ALU ที่สุด ทำงานไวที่สุด
  5. **Static RAM (SRAM):** [0 แต้ม] แรมความเร็วสูงที่ไม่ต้องคอยเติมไฟ (Refresh) มักทำเป็น Cache
  6. **Dynamic RAM (DRAM):** [0 แต้ม] แรมความจุสูงที่มีราคาประหยัด ต้องคอยเติมประจุไฟฟ้าตลอดเวลา
- **1 ดาว (2 ใบ):**
  7. **Data Structure Basics (โครงสร้างข้อมูลเบื้องต้น):** [1 แต้ม] การใช้งาน Array, List, และ Stack พื้นฐานเพื่อจัดเรียงข้อมูล
  8. **Cache L1 (แคชระดับ 1):** [1 แต้ม] หน่วยความจำความเร็วแสงที่ซ่อนอยู่ในตัวซีพียู

### Tier 2 (6 ใบ)
- **1 ดาว (2 ใบ):**
  9. **Memory Allocation / Malloc (ระบบจองหน่วยความจำ):** [1 แต้ม] ซอฟต์แวร์ฟังก์ชันระดับระบบปฏิบัติการสำหรับขอพื้นที่ RAM ไปให้โปรแกรมใช้งาน
  10. **Garbage Collection (ตัวเก็บขยะ):** [1 แต้ม] เทคนิคในภาษาอย่าง Java/C# ที่คอยตามเก็บคืนพื้นที่ความจำที่ไม่ได้ใช้งานแล้วแบบอัตโนมัติ
- **2 ดาว (2 ใบ):**
  11. **Virtual Memory (หน่วยความจำเสมือน):** [2 แต้ม] ระบบที่ทำการหลอกแอปพลิเคชันว่ามี RAM มหาศาลด้วยการหยิบยืมพื้นที่จากฮาร์ดดิสก์
  12. **Paging System (ระบบแบ่งหน้าข้อมูล):** [2 แต้ม] การหั่นข้อมูลเป็นหน้าๆ สลับเข้าออก RAM เพื่อแก้ปัญหาข้อมูลขาดตอน
- **3 ดาว (2 ใบ):**
  13. **Cache L2/L3 Architecture (สถาปัตยกรรมแคชรวม):** [4 แต้ม] การวางแผนแคชขนาดใหญ่แชร์ข้อมูลให้หลายแกนซีพียูอย่างฉลาด
  14. **Buffer Overflow Protection (ป้องกันบัฟเฟอร์ล้น):** [4 แต้ม] ระบบการวางขอบเขตความจำให้ปลอดภัย ป้องกันข้อมูลไหลไปทับส่วนอื่น

### Tier 3 (4 ใบ)
- **3 ดาว (1 ใบ):**
  15. **Shared Memory Model (โมเดลหน่วยความจำร่วม):** [4 แต้ม] พื้นที่หน่วยความจำกลางที่ให้ต่างแอปพลิเคชันสามารถคุยกันได้
- **4 ดาว (2 ใบ):**
  16. **In-Memory Database (ฐานข้อมูลในแรม):** [5 แต้ม] โครงสร้างอย่างเช่น Redis ที่เก็บฐานข้อมูลทั้งหมดบน RAM ทำให้เรียกใช้เร็วมากในระดับมิลลิวินาที
  17. **Distributed Cache (แคชแบบกระจายตัว):** [5 แต้ม] เชื่อมต่อแรมในเซิร์ฟเวอร์หลายเครื่องมาตอบโจทย์ระบบขนาดใหญ่
- **5 ดาว (1 ใบ):**
  18. **Holographic/Quantum Memory (หน่วยความจำล้ำยุค):** [7 แต้ม] นวัตกรรมเก็บข้อมูลในสถานะควอนตัมและสามมิติ ทำงานข้ามมิติได้

## ⬜ การ์ด Storage สีขาว (หมวดจัดเก็บข้อมูลถาวร ไฟล์และฐานข้อมูล)
*เน้นไปที่ระบบฐานข้อมูล (Database), โครงสร้างไฟล์, และศูนย์กักเก็บข้อมูล*

### Tier 1 (8 ใบ)
- **0 ดาว (6 ใบ):**
  1. **Text File & Directory (ไฟล์และไดเรกทอรี):** [0 แต้ม] การจัดเก็บข้อมูลเป็นลำดับขั้น แฟ้มและกล่องพื้นฐานที่สุด
  2. **Magnetic Disk / HDD (ฮาร์ดดิสก์แม่เหล็ก):** [0 แต้ม] การเก็บข้อมูลรอยผ่านจานแม่เหล็ก ความจุสูง ราคาคุ้มค่า
  3. **Solid State Drive / SSD (หน่วยเก็บข้อมูลโซลิดสเตต):** [0 แต้ม] การเก็บข้อมูลผ่าน Flash Memory ความเร็วสูงขึ้น ไม่มีชิ้นส่วนหมุน
  4. **Basic Query (ชุดคำสั่งค้นหาเบื้องต้น):** [0 แต้ม] ระบบค้นหาและบันทึกข้อมูลง่ายๆ ในซอฟต์แวร์
  5. **Data Types (ชนิดของไฟล์):** [0 แต้ม] โครงสร้างของข้อมูล เช่น ข้อความตัวเลข ไปจนถึงไฟล์รูปภาพและวิดีโอ
  6. **FAT/NTFS Formatter (ตัวฟอร์แมตไฟล์เบื้องต้น):** [0 แต้ม] สถาปัตยกรรมการแบ่งพาร์ติชันดิสก์ยุคเริ่มแรก
- **1 ดาว (2 ใบ):**
  7. **Relational Data Base (ฐานข้อมูลเชิงสัมพันธ์แบบง่าย):** [1 แต้ม] การโยงความสัมพันธ์ของตาราง 2 แบบ
  8. **B-Tree Indexing (ดัชนีการค้นหาบีทรี):** [1 แต้ม] เทคนิคการสร้างโครงสร้างต้นไม้ลดระยะเวลาการควานหาข้อมูลในดิสก์

### Tier 2 (6 ใบ)
- **1 ดาว (2 ใบ):**
  9. **JSON/XML Formatting (โครงสร้างข้อมูลแบบมาตรฐาน):** [1 แต้ม] การจัดระบบข้อมูลสำหรับสื่อสารระหว่างแอปพลิเคชัน
  10. **Data Compression (บีบอัดข้อมูล):** [1 แต้ม] อัลกอริทึมการบีบไฟล์ให้เล็กลงเพื่อประหยัดพื้นที่จัดเก็บ
- **2 ดาว (2 ใบ):**
  11. **SQL Database (ฐานข้อมูล SQL):** [2 แต้ม] ระบบฐานข้อมูลที่เข้มงวดและน่าเชื่อถือสูง ใช้มากในธุรกรรมทางการเงิน
  12. **NoSQL Database (ฐานข้อมูลแบบกระจายโครงสร้าง):** [2 แต้ม] ระบบที่รองรับโครงสร้างไม่แน่นอน ไวและยืดหยุ่น เช่น MongoDB
- **3 ดาว (2 ใบ):**
  13. **RAID Array (ระบบดิสก์จำลองเชิงซ้อน):** [4 แต้ม] การต่อฮาร์ดดิสก์หลายตัวเข้าด้วยกันเพื่อบูสต์ความเร็วและอัพเกรดระบบสำรอง
  14. **Automated Backup & Recovery (กู้คืนข้อมูลง่ายดาย):** [4 แต้ม] โปรโตคอลเซฟข้อมูลสำคัญไว้สำรองเมื่อเกิดเหตุไม่คาดฝัน

### Tier 3 (4 ใบ)
- **3 ดาว (1 ใบ):**
  15. **Data Warehouse (คลังข้อมูลองค์กร):** [4 แต้ม] แหล่งรวมและวิเคราะห์ข้อมูลดิบทั้งหมดของบริษัทเพื่อการสรุปผลเชิงลึก
- **4 ดาว (2 ใบ):**
  16. **Distributed File System / HDFS (ระบบไฟล์กระจายตัวศูนย์):** [5 แต้ม] โครงสร้างไฟล์ที่แหวะกระจายข้อมูลไปเก็บซ่อนในหลายๆ เซิร์ฟเวอร์พร้อมกัน (เช่น Hadoop)
  17. **Cloud Object Storage (พื้นที่ความจำคลาวด์มหาศาล):** [5 แต้ม] บริการรับฝากไฟล์บนคลาวด์ขนาดมหาศาลเช่น Amazon S3
- **5 ดาว (1 ใบ):**
  18. **Big Data Data Center (ศูนย์บิ๊กดาต้าเลค):** [7 แต้ม] นวัตกรรมระดับมหภาค ศูนย์ข้อมูลขนาดบิ๊กดาต้าที่รันระบบ Data Mining เพื่อวิเคราะห์พฤติกรรมมนุษย์
