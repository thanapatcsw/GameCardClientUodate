# 2. วัตถุประสงค์ของโครงงาน (Objective)

2.1 เพื่อพัฒนาโมบายแอปพลิเคชันเกมการ์ดเชิงศึกษาที่จำลองกระบวนการบริหารจัดการทรัพยากรและการประกอบโมดูลซอฟต์แวร์ด้วย Unity

2.2 พัฒนาระบบตรรกะเชิงกฎเกณฑ์ (Rule-Based System) เพื่อควบคุมตัวแทนผู้เล่นคอมพิวเตอร์ (Bot) ให้สามารถเล่นร่วมกับผู้เล่นในโหมดผู้เล่นคนเดียว (Single Player) ได้อย่างมีประสิทธิภาพ

2.3 เพื่อพัฒนาและออกแบบระบบฐานข้อมูลให้รองรับกับการใช้งานของผู้ใช้ในระบบเกมการ์ดเสริมสร้างแนวคิด
