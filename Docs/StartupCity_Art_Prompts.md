# Startup City - คัมภีร์พร้อมท์ภาพประกอบการ์ด (ฉบับภาษาไทย 90 ใบ)

เอกสารนี้รวบรวมคำอธิบายรายละเอียดภาพประกอบของการ์ดทั้ง 90 ใบแบบเจาะลึก เพื่อนำไปใช้เป็นบรีฟงานอาร์ตเวิร์ค หรือก๊อปปี้ไปเป็นคีย์เวิร์ดตั้งต้นให้ AI สร้างภาพได้ตรงตามวิสัยทัศน์เกมของคุณครับ

---
### 🟩 สไตล์ภาพรวมที่ต้องมีในทุกๆ การ์ด (Base Art Direction):
**"ภาพไอคอนเวกเตอร์แบบแบน (Flat Vector) สไตล์มินิมอล, ลายเส้นวาดสีขาวที่สว่างและคมกริบ (Sleek white line art), จัดวางกราฟิกอยู่กึ่งกลางภาพแบบสมมาตรพอดี, ปล่อยพื้นที่ว่าง (Negative space) รอบๆ ให้โล่งกว้างสุดๆ, พื้นหลังเป็นสีทึบล้วน (Solid color) ตามหมวด, สไตล์งานศิลปะแนว UI ของเกมไพ่คลาสสิก, เน้นรูปทรงเรขาคณิต, ไม่มีการไล่สี (No gradients), ไม่มีตัวหนังสือหรือตัวอักษรผสม, ไม่มีองค์ประกอบแบบสามมิติ (No 3D)"**

## 🟥 หมวดประมวลผล (CPU - สีแดง)
*สีพื้นหลัง: สีแดงเข้มทึบล้วน (Pure Solid Dark Red / Crimson #4a0a10)*

### Tier 1
1. **Transistor:** ภาพสัญลักษณ์อิเล็กทรอนิกส์ของทรานซิสเตอร์ 3 ขาที่เรียบง่ายเด่นชัด
2. **Logic Gate:** ภาพสัญลักษณ์ลอจิกเกตแบบ AND (รูปทรงตัว D) มีเส้นรับข้อมูลเข้าหน้าสองเส้นและเส้นส่งออกหลังหนึ่งเส้น
3. **ALU:** ภาพสัญลักษณ์เรขาคณิตสุดคลาสสิกของหน่วยคำนวณ ALU รูปทรงสามเหลี่ยมหัวคว่ำคล้ายตัว V
4. **Clock Signal:** ภาพกราฟจังหวะสัญญาณนาฬิกาดิจิทัลแบบลูกคลื่นวงจรเหลี่ยมที่ต่อเนื่องกัน
5. **Assembler:** ภาพวงล้อฟันเฟืองจักรกลล้ำยุค ตรงกลางเฟืองมีรูปทรงนามธรรมเรียบๆ ที่สื่อถึงชุดรหัสเครื่อง
6. **Syntax Structure:** ภาพสัญลักษณ์วงเล็บปีกกาอัลกอริทึม "{ }" ขนาดใหญ่ที่ดูโฉบเฉี่ยว
7. **Microprocessor:** ภาพแผงวงจรไมโครชิปสี่เหลี่ยมจัตุรัสแบบดั้งเดิม มีแง่งขาเชื่อมต่อแผ่ออกมาทั้ง 4 ด้าน
8. **Pipelining:** ภาพลูกศรเส้นประ 3 เส้นที่วิ่งขนานไปข้างหน้าพร้อมๆ กันในรูปแบบสายพานลำเลียง

### Tier 2
9. **Task Scheduler:** ภาพไอคอนนามธรรมของปฏิทินหรือตารางกริดที่กำลังถูกขีดฆ่าเช็คลิสต์งาน
10. **Interrupt Handler:** ภาพสายฟ้าฟาดอันแหลมคมที่ผ่าเส้นตรงแนวนอนขาดออกจากกัน
11. **Multi-Core:** ภาพช่องสี่เหลี่ยมจัตุรัสใหญ่ 1 ชิ้น ที่ภายในถูกผ่าแบ่งออกเป็นตารางลูกบาศก์ 4 ช่องย่อยเท่าๆ กัน
12. **Multithreading:** ภาพเส้นใยพลังงาน 3 เส้นที่พันเกลียวหรือพุ่งคู่ขนานกันไปอย่างพริ้วไหว
13. **Hyper-Threading:** ภาพเส้นด้ายแม่ 1 เส้นที่พุ่งมาแล้วแตกแขนงออกเป็นเส้นคู่ขนานเสมือน 2 เส้น
14. **GPU:** ภาพใบพัดระบายความร้อนของชิปการ์ดจอหลายแฉก โดยมองจากมุมบน (Top-down view)

### Tier 3
15. **Server Farm CPU:** ภาพตู้เซิร์ฟเวอร์เรียงซ้อนกัน 3 ชั้น มีจุดไฟสถานะเล็กๆ ส่องสว่างที่หน้าตู้
16. **Neural Processing Unit (NPU):** ภาพโครงข่ายประสาทเทียมที่มีจุดโหนดเชื่อมโยงกันจนเกิดเป็นรูปทรงสมองจักรกลแบบมินิมอล
17. **Quantum Processor (QPU):** ภาพแกนกลางประมวลผลควอนตัมแห่งอนาคต ล้อมรอบด้วยวงโคจรของอิเล็กตรอนที่พันเป็นวงรีรูปไข่
18. **Supercomputer Cluster:** ภาพโหนดคอมพิวเตอร์แบบไร้ศูนย์กลางจำนวนมหาศาลที่โยงเส้นเข้าหากันและพุ่งระเบิดแสงไปยังจุดศูนย์กลางเดียว

## 🟦 หมวดหน่วยความจำ (RAM - สีฟ้า)
*สีพื้นหลัง: สีฟ้าน้ำทะเลลึกหรือกรมท่าทึบล้วน (Pure Solid Deep Blue / Cyan #0a1b30)*

### Tier 1
19. **Bit:** ภาพกล่องบล็อกข้อมูลสี่เหลี่ยมจิ๋วที่มีตัวเลข '1' ทรงเรขาคณิตอยู่ข้างใน
20. **Variable:** ภาพกล่องลิ้นชักข้อมูลดิจิทัลหน้าเรียบๆ ที่ถูกปิดสนิท 1 ชั้น
21. **Pointer:** ภาพลูกศรทรงแหลมคมชี้ดิ่งลงไปที่โหนดพิกัดวงกลมเล็กๆ ด่านล่าง
22. **CPU Registers:** ภาพแถบข้อมูลแนวนอน 3 แถบเรียงซ้อนทับกันเป็นชั้นๆ อย่างมีระเบียบ
23. **Static RAM / SRAM:** ภาพแผ่นวงจรหน่วยความจำรูปสี่เหลี่ยมผืนผ้าแนวนอน มีดีเทลจุดเชื่อมต่อเล็กๆ รอบๆ
24. **Dynamic RAM / DRAM:** ภาพแผงแรมแท่งทรงตั้งเรียวยาวสุดคลาสสิก ถูกแบ่งช่องชิปความจำย่อยเป็น 4 บล็อก
25. **Data Structure Basics:** ภาพบล็อกสี่เหลี่ยม 3 ชิ้นเชื่อมต่อติดกันอย่างเป็นตรรกะและมีแบบแผน
26. **Cache Memory L1:** ภาพชิปซีพียูสี่เหลี่ยม โดยมีช่องเล็กๆ เรืองแสงถูกจำกัดขอบเขตไว้ที่มุมซ้ายล่างเพียงมุมเดียว

### Tier 2
27. **Memory Allocation:** ภาพตารางหน่วยความจำ 3x3 ช่อง โดยมีลูกบาศก์ตรงกลาง 4 ช่องเท่านั้นที่ถูกขีดไฮไลต์จองพื้นที่
28. **Garbage Collection:** ภาพบล็อกข้อมูลดิจิทัลที่กำลังละลายหรือร่วงหล่นลงในสัญลักษณ์ถังขยะรีไซเคิล
29. **Virtual Memory:** ภาพรูปทรงก้อนเมฆดิจิทัลที่ผูกโยงสายรับส่งข้อมูลตรงเข้ากับแท่งแรมดั้งเดิม
30. **Paging & Swapping:** ภาพแผ่นเอกสารนามธรรม 2 แผ่นกำลังสลับสับเปลี่ยนตำแหน่งกัน โดยมีลูกศรคู่โค้งวาดล้อมรอบ
31. **L2/L3 Cache Architecture:** ภาพฮับหน่วยความจำขนาดใหญ่ตรงกลางที่แตกลายเส้นรับส่งข้อมูลกระจายออกไปยังชิปขนาดเล็ก 4 ตัวรอบทิศ
32. **Buffer Overflow Protection:** ภาพโล่ป้องกันเหล็กกล้าแบนราบที่ซ้อนทับอยู่บนกล่องสี่เหลี่ยมที่กำลังมีข้อมูลล้นทะลักออกจากขอบ

### Tier 3
33. **Shared Memory Model:** ภาพหน้าต่างแอปพลิเคชัน 2 บานที่แยกกัน แต่มีเส้นโยงสะพานโค้งเชื่อมข้อมูลมาลงยังบ่อตรงกลางบ่อเดียวกัน
34. **In-Memory Database:** ภาพถังฐานข้อมูลทรงกระบอกคลาสสิก ตั้งตระหง่านสมดุลอยู่บนแผงวงจรชิปแรมแบนๆ
35. **Distributed Cache:** ภาพเครือข่ายสามเหลี่ยมของชิปความจำ 3 ตัวที่โยงเส้นคุยกันเป็นลูปวงกลมไร้รอยต่อ
36. **Holographic Memory:** ภาพปริซึมคริสตัลโฮโลแกรม 5 มิติที่กำลังแผ่ลำแสงออปติคอลกระจายออกไปรอบทิศทาง

## ⬜ หมวดพื้นที่เก็บข้อมูล (Storage - สีขาว/เงิน)
*สีพื้นหลัง: สีเทาเข้มเนื้อด้านทึบล้วน (Pure Solid Dark Grey / Slate #1a1a1a)*

### Tier 1
37. **Text File:** ภาพหน้ากระดาษเอกสารเดี่ยวๆ ที่มีเส้นขีดแนวนอนเป็นตัวหนังสือ 3 บรรทัด
38. **Directory:** ภาพโครงร่างซิลลูเอทของแฟ้มโฟลเดอร์สำนักงานที่แบนราบที่สุด
39. **Basic Query:** ภาพแว่นขยายที่กำลังสอดส่องและสอบสวนไอคอนถังฐานข้อมูลจิ๋ว
40. **Magnetic Hard Disk (HDD):** ภาพแผ่นจานแม่เหล็กวงกลมที่มีเข็มหัวอ่าน (Actuator arm) จักรกลพาดทับตรงกลาง
41. **Solid State Drive (SSD):** ภาพกรอบสี่เหลี่ยมหุ้มไดรฟ์ SSD พร้อมลายเส้นแผงวงจรพิมพ์ลายสวยงาม
42. **FAT Formatter:** ภาพแผ่นดิสก์ฟลอปปี้ขนาด 3.5 นิ้วรุ่นเก๋าที่คุ้นเคย
43. **Relational Database:** ภาพตารางกริดข้อมูล 2 ตารางที่ถูกโยงเข้าหากันด้วยลูกศรความสัมพันธ์ที่แม่นยำ
44. **B-Tree Indexing:** ภาพแผนผังโครงสร้างต้นไม้ที่แผ่กิ่งก้านคว่ำลง

### Tier 2
45. **JSON Formatting:** ภาพวงเล็บปีกกาครอบเครื่องหมายโคลอน "{ : }" วางไว้กึ่งกลาง
46. **Data Compression:** ภาพลูกศรสองหัวที่ชี้ออกด้านนอก แต่ถูกบล็อกเหล็กแนวตั้งสองแท่งบีบอัดดันกลับเข้ามา
47. **SQL Standards:** ภาพถังฐานข้อมูลทรงกระบอกที่แข็งแกร่ง ถูกป้องกันด้วยแม่กุญแจขนาดใหญ่หรือตราประทับขี้ผึ้ง
48. **NoSQL Database:** ภาพถังฐานข้อมูลที่เปลือกแตกออกเป็นชิ้นส่วน หรือย่อยสลายกลายเป็นโครงสร้างโพลิกอนอิสระ
49. **RAID Array:** ภาพถังฐานข้อมูล 3 ถังยืนเรียงเคียงข้างกันและแชร์ฐานเส้นเชื่อมต่อเส้นเดียวกันทั้งหมด
50. **Snapshot Recovery:** ภาพลูกศรชี้ทวนเข็มนาฬิกาที่กวาดโค้งย้อนเวลากลับไปเหนือรูปสัญลักษณ์ฮาร์ดไดรฟ์

### Tier 3
51. **Data Warehouse:** ภาพอาคารคลังสินค้าขนาดมหึมาหรือสถาปัตยกรรมโกดังยักษ์ที่มีสัญลักษณ์ DB ซ้อนอยู่
52. **Distributed File System:** ภาพหยดน้ำเรขาคณิต 5 หยดที่กระจัดกระจาย แต่ร้อยลวดเส้นใยแมงมุมเข้าหากันกลายเป็นรูปก้อนเมฆ
53. **Cloud Object Storage:** ภาพก้อนเมฆดิจิทัลที่นุ่มนวลกำลังโอบอุ้มกล่องพัสดุเก็บข้อมูลรูปทรงเรขาคณิตที่เบาหวิว
54. **Big Data Data Center:** ภาพศูนย์เก็บข้อมูลขนาดมโหฬารที่เป็นแท่งตระหง่าน แผ่กระจายคลื่นสัญญาณเรดาร์ออกไปกว้างจนเกือบหลุดกรอบภาพ

## 🟪 หมวดเครือข่ายอินเทอร์เน็ต (Network - สีม่วงเข้ม)
*สีพื้นหลัง: สีม่วงเข้มลึกลับทึบล้วน (Pure Solid Deep Purple #1a0526)*

### Tier 1
55. **LAN Cable:** ภาพหน้าตัดตรงของหัวแจ็คเสียบสายแลน LAN (RJ45) แบบแบนๆ
56. **MAC Address:** ภาพป้ายบาร์โค้ดฮาร์ดแวร์ที่แสดงบล็อกตัวเลขฐานสิบหกแบบนามธรรม
57. **Network Hub:** ภาพกล่องโหนดสี่เหลี่ยมผืนผ้าตรงกลางที่ปัดปล่อยเส้นสัญญาณกระจายออกไปยัง 4 พอร์ตรับขาออกอย่างเท่าเทียม
58. **IP Address:** ภาพหมุดปักหมุด Location Map คว่ำลงชี้เป้าหมายไปที่ลูกโลกดิจิทัล
59. **Binary Packet:** ภาพกล่องพัสดุผูกห่อด้วยเกราะคลื่นสัญญาณจราจรทางเน็ต
60. **Modem:** ภาพเร้าเตอร์ทรงตั้งที่แผ่คลื่นความถี่ Wi-Fi โค้งขึ้นไปเบื้องบนเป็น 3 ระดับ
61. **Switch:** ภาพกล่องเครือข่ายอัจฉริยะที่มีเส้นทางลูกศรวิ่งไขว้ทแยงหรือหักเลี้ยวสับรางกันฝังอยู่ภายใน
62. **HTTP Protocol:** ภาพตัวหนังสือพิมพ์ใหญ่คำว่า HTTP ดัดโค้งไปตามขอบบนของโครงลูกโลกเส้นใยลวด

### Tier 2
63. **DHCP Server:** ภาพเครื่องจ่ายตั๋วอัตโนมัติที่กำลังหมุนพิมพ์แผ่นป้ายทะเบียน IP ออกมา
64. **DNS Registry:** ภาพสมุดจดที่อยู่ปกแข็งกระด้าง ถูกประทับตราสัญลักษณ์ลูกโลกไว้กลางหน้าปก
65. **Router:** ภาพลูกโลกทรงกลมตรงกลางที่ล้อมรอบด้วยลูกศรเครือข่ายฉวัดเฉวียนวิ่งวนเข้าออกเป็นลูป
66. **TCP/IP Protocol:** ภาพตรายางประทับอนุมัติอย่างเป็นทางการ (Approved) ประทับลงบนกล่องพัสดุรับส่งข้อมูลระดับเน็ตเวิร์ก
67. **Load Balancer:** ภาพตาชั่งทองเหลืองสมดุลเป๊ะ ที่สองฟากกวาดรับน้ำหนักบล็อกตู้เซิร์ฟเวอร์ยักษ์ฝั่งละเท่าๆ กัน
68. **Web Socket:** ภาพท่อแก้วโปร่งใส 2 เส้นตีคู่ขนานกัน โดยมีลำแสงเลเซอร์หนาแน่นพุ่งสวนทางกันอยู่ภายในตลอดเวลา

### Tier 3
69. **CDN (Content Delivery Network):** ภาพลูกโลกส่วนกลางที่ถูกดาวเทียมวงโคจรจิ๋ว 5 ดวงบินตีกรอบล້ອມเข้าประชิดสุดใจ
70. **Cloud Mesh Network:** ภาพโครงข่ายรังผึ้งหกเหลี่ยม 3 ชิ้นเชื่อมประสานกันอย่างเหนียวแน่นและแตกหักไม่ได้
71. **Optical Fiber Backbone:** ภาพเคเบิลสายใยแก้วนำแสงตรงกลางที่เปล่งแสงคลื่นไซน์ (Sine wave) หนาทะลุทะลวง
72. **IPFS (InterPlanetary File System):** ภาพดาวเคราะห์จิ๋ว 3 ดวงเรียงกันแบบกึ่งกระจัดกระจายแต่ร้อยโซ่ตรวนอวกาศติดกันทั้งหมด

## 🟨 หมวดความปลอดภัย (Security - สีเหลือง/ทอง)
*สีพื้นหลัง: สีทองทึบหม่นหรือสีอำพันทึบล้วน (Pure Solid Dark Gold / Amber #302606)*

### Tier 1
73. **Username Password:** ภาพลูกกุญแจฟันปลาแข็งๆ กำลังเสียบปลดล็อคเข้ากับซิลลูเอทรูปหัวคนดิจิทัล
74. **Antivirus:** ภาพโล่เหล็กปราการปราบศัตรู ที่หน้าโล่มีเครื่องหมายกากบาทสาธารณสุขหรือบวกการแพทย์ (Health)
75. **Plaintext:** ภาพหน้ากระดาษเอกสารที่เปิดเผยหมดเปลือก ไม่มีกรอบป้องกันและถูกแสตมป์ลายหัวกะโหลกอันตรายแบบจางๆ
76. **Access Token:** ภาพบัตรประจำตัวตัวแทนแบนราบติดตราประทับพร้อมสายคล้องคอ V.I.P. ดิจิทัล
77. **System Logs:** ภาพม้วนกระดาษใบเสร็จที่หลุดไหลออกมาไม่มีวันสิ้นสุด โชว์ประวัติรอยเท้าคอมพิวเตอร์
78. **Port Blocking:** ภาพรูกุญแจที่ถูกสั่งปิดตายด้วยตรากากบาทสีแดงหรือเส้นคาดทับฉับพลัน
79. **Salting Password:** ภาพขวดเกลือเข้ารหัสหรือขวดพริกไทย กำลังโรยจุดคริปโตกราฟีลงในกล่องเครื่องหมายรหัสผ่าน
80. **Symmetric Encryption:** ภาพแม่กุญแจยักษ์หนาหนัก ที่ถูกห่อหุ้มคลุมอย่างสมบูรณ์รอบทิศด้วยวงแหวนรหัสเลขฐานสอง

### Tier 2
81. **Asymmetric Key (RSA):** ภาพแม่กุญแจที่ถูกแบ่งผ่าครึ่งซีกอย่างชัดเจน ครึ่งซ้ายสีทึบ ครึ่งขวาสีโปร่งสว่าง (กุญแจเข้าคู่)
82. **Firewall Rules:** ภาพกำแพงอิฐตั้งตระหง่านที่มีลวดลายเปลวไฟสุดวิจิตรคอยเผาดักหน้าประตูแพ็กเกจ
83. **HTTPS / SSL:** ภาพแม่กุญแจนิรภัยความมั่นคงสูง ฝังผนึกติดเป็นเนื้อเดียวกันกับด้านหน้าของแถบ URL เว็บไซต์
84. **DDoS Mitigation:** ภาพโล่พลังงานเวกเตอร์รูปร่างคางหมู กำลังกระแทกสะท้อนคลื่นโจมตีดุร้าย 3 ระลอกกลับไป
85. **Two-Factor Auth (2FA):** ภาพหน้าจอโทรศัพท์สมาร์ตโฟนแสดงแม่กุญแจปลอดภัย ซ้อนคู่ขนานกับข้อความสายฟ้า SMS แจ้งเตือนเข้า
86. **Penetration Testing (Pentest):** ภาพหมวกพิทักษ์เครือข่ายปีกกว้างทรงเฟโดรา (Hacker's White Hat) วางไว้บนโต๊ะดิจิทัลแบนราบ

### Tier 3
87. **JWT (JSON Web Token):** ภาพเหรียญตราคริปโตหรือตัวเหรียญโทเคนรูปวงกลมลอยฟ้าพร้อมลงตราประทับรับรองที่แข็งแกร่ง
88. **Blockchain Ledger:** ภาพลูกบาศก์ก้อนบัญชี 3 กล่องหนาตึ้บที่ถูกล่ามตรวนโซ่ห่วงขนาดใหญ่ผูกตายมัดรวมกันตลอดกาล
89. **End-to-End Encryption:** ภาพโทรศัพท์มือถือ 2 เครื่องหันหน้าเข้าหากัน มีท่อสายตรงวิ่งเชื่อมถึงกันและถูกล็อคด้วยแม่กุญแจยักษ์ไว้กึ่งกลางสาย
90. **Zero Trust Architecture:** ภาพเครื่องสแกนเรดาร์ไร้ปราณีหรือดวงตาจับผิด ที่ขีดเส้นสะกัดกั้นแบนรอยเท้าทุกชนิดที่เดินผ่านเข้าใกล้
