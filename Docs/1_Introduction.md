# 1. ที่มาและความสำคัญ (Introduction)

ในปัจจุบัน การเรียนรู้ด้านวิทยาการคอมพิวเตอร์ โดยเฉพาะด้านสถาปัตยกรรมระบบและการพัฒนาซอฟต์แวร์ (Software Development) เป็นเนื้อหาพื้นฐานที่มีความสำคัญต่อนักศึกษา อีกทั้งเนื้อหาดังกล่าวมีความหลากหลาย การเรียนรู้โดยอาศัยการท่องจำเพียงอย่างเดียวอาจก่อให้เกิดความน่าเบื่อและทำให้ขาดแรงจูงใจในการศึกษา 

ด้วยเหตุนี้ ผู้จัดทำจึงเลือกแทรกคำถามระหว่างเล่นเกมเพื่อเสริมสร้างแรงจูงใจ และมีแนวคิดในการนำกลไกจากเกม Splendor (เกมค้าเพชร) ซึ่งเป็นเกมแนวบริหารจัดการทรัพยากรที่ประสบความสำเร็จสูง มาประยุกต์ใช้เป็นเครื่องมือการเรียนรู้ในรูปแบบเกมการ์ดดิจิทัล โดยเปลี่ยนกระบวนการสะสมเพชรมาเป็นการจัดการทรัพยากรระบบคอมพิวเตอร์ ได้แก่ CPU, RAM, Network, Storage และ Security เพื่อนำไปแลกซื้อการ์ดที่ทำเป็นธีมคอมพิวเตอร์

ยิ่งไปกว่านั้น เพื่อยกระดับให้ซอฟต์แวร์จำลองการศึกษาพัฒนาก้าวหน้าขึ้นเป็นระบบเกมออนไลน์ที่ใช้งานได้จริง โครงงานนี้จึงนำสถาปัตยกรรมซอฟต์แวร์แบบ Full-Stack มาประยุกต์ใช้ในการทำงานจริง โดยตัวเกมพัฒนาขึ้นด้วย Unity เชื่อมต่อระบบเครือข่ายเรียลไทม์ผ่าน Photon Fusion (Shared Mode) รองรับผู้เล่น 2-4 คน ควบคู่กับฐานข้อมูลคลาวด์หลังบ้านอย่าง Supabase (PostgreSQL) สำหรับจัดการสถิติ ค่า MMR และระบบคิวจับคู่อัตโนมัติ (Auto Matchmaking) อีกทั้งยังมีระบบควบคุมตัวแทนผู้เล่นคอมพิวเตอร์ด้วยตรรกะเชิงกฎเกณฑ์ (Rule-Based Bot System) เพื่อเล่นกับผู้เล่นในโหมด Single Player และบูรณาการระบบควิซประเมินความรู้คอมพิวเตอร์สอดแทรกระหว่างเล่น (Interventional Quiz) ทุกๆ 3 รอบการเล่น เพื่อผลักดันให้แอปพลิเคชันนี้ทำหน้าที่เป็นสื่อการเรียนรู้เชิงปฏิสัมพันธ์ที่เปลี่ยนแนวคิดเชิงทฤษฎีที่เข้าใจยากให้กลายเป็นแบบจำลองกิจกรรมที่สนุกสนานและจับต้องได้จริง
