# ฐานข้อมูลการ์ดเกม Startup City (ฉบับสมบูรณ์ 100% - 90 ใบ)

ระบบการ์ดทั้งหมด 5 หมวดสี หมวดละ 18 ใบ รวม 90 ใบ สอดคล้องกับแนวคิดวิทยาการคอมพิวเตอร์ แต่ละใบมีการเจาะลึกคำอธิบายที่พร้อมใช้นำไปทำ UI Tooltip ภายในเกม และโน้ตสำหรับการทำอาร์ตเวิร์กให้ออกมาเป็นแนว **Minimalist** เหมาะกับเกมมือถือ (หน้าจอเล็ก ดูง่าย เข้าใจทันที ไม่รก)

### แนวทางศิลปะภาพประกอบการ์ด (Art Direction)
- **สไตล์:** Flat Design หรือ Line Art แบบมินิมอล พื้นหลังสีทึบหรือสีพาสเทลตามสีของการ์ด (แดง, ฟ้า, ดำ, ขาว, เหลือง) ไอคอนตรงกลางเด่นชัด
- **การจัดวาง:** มี Negative space (พื้นที่ว่าง) รอบๆ เยอะๆ เพื่อให้ผู้เล่นกวาดตามองแล้วรู้ทันทีว่าคือหมวดไหน

## 🟥 หมวดประมวลผล (CPU - สีแดง)
*โฟกัส: พลังงานสมอง กลไก สถาปัตยกรรมการแปรผล*

### Tier 1 (8 ใบ)
- **[0 ดาว / 0 แต้ม] Transistor (ทรานซิสเตอร์)**
  - **คำอธิบาย UI:** สวิตช์อิเล็กทรอนิกส์ขนาดจิ๋ว หน่วยย่อยและรากฐานที่ลึกที่สุดของซีพียู ทำหน้าที่คอยเปิด-ปิดกระแสไฟฟ้าเพื่อสร้างสถานะลอจิก 0 และ 1
  - **ภาพประกอบ:** สัญลักษณ์วงจรทรานซิสเตอร์ 3 ขาแบบเส้นบาง (Line art) บนพื้นแดง
- **[0 ดาว / 0 แต้ม] Logic Gate (ลอจิกเกต)**
  - **คำอธิบาย UI:** วงจรตรรกะพื้นฐานที่รับสัญญาณไฟฟ้านำมาประมวลผลตามกฎเกณฑ์ (AND, OR, NOT) เพื่อเปรียบเทียบเงื่อนไขก่อนส่งต่อ
  - **ภาพประกอบ:** รูปทรงตัว D (ครึ่งวงกลมของสัญลักษณ์ AND Gate) 
- **[0 ดาว / 0 แต้ม] ALU (Arithmetic Logic Unit)**
  - **คำอธิบาย UI:** แกนหลักแห่งคณิตศาสตร์ในคอมพิวเตอร์ รับคำสั่งมาบวกลบคูณหาร รวมถึงเปรียบเทียบตรรกะซับซ้อนอย่างรวดเร็ว
  - **ภาพประกอบ:** รูปทรงเรขาคณิตสามเหลี่ยมรูปตัว V สัญลักษณ์คลาสสิกของกระบวนการ ALU
- **[0 ดาว / 0 แต้ม] Clock Signal (สัญญาณนาฬิกา)**
  - **คำอธิบาย UI:** ตัวกำกับจังหวะชีพจรของซีพียู ทุกความถี่ที่เต้นคือเข็มทิศให้ทุกภาคส่วนของโปรเซสเซอร์ทำงานเข้าขากันพอดีเป๊ะ
  - **ภาพประกอบ:** กราฟสัญญาณลูกคลื่นสี่เหลี่ยม (Square Wave) แบบมินิมอล
- **[0 ดาว / 0 แต้ม] Assembler (แอสเซมเบลอร์)**
  - **คำอธิบาย UI:** โปรแกรมแปลภาษาที่เปลี่ยนชุดตัวอักษรภาษามนุษย์ขั้นต่ำ ให้กลายเป็นรหัสภาษาเครื่องที่ซีพียูเข้าใจและรันได้
  - **ภาพประกอบ:** ไอคอนรูปฟันเฟืองที่มีรหัส '01' อยู่ตรงกลาง
- **[0 ดาว / 0 แต้ม] Syntax Structure (โครงสร้างไวยากรณ์)**
  - **คำอธิบาย UI:** การวางกฎเกณฑ์บล็อกคำสั่งแบบมีเงื่อนไข เช่น if-else หรือลูป เพื่อกำหนดตรรกะและเส้นทางการตัดสินใจของแอปพลิเคชัน
  - **ภาพประกอบ:** ไอคอนวงเล็บปีกกา `{ }` เส้นหนาสีขาว
- **[1 ดาว / 1 แต้ม] Microprocessor (ไมโครโปรเซสเซอร์)**
  - **คำอธิบาย UI:** วงจรรวมที่ปฏิวัติวงการ นำซีพียูที่เคยขนาดเท่าตู้เย็นมาย่อส่วนลงไปอยู่ในชิปซิลิคอนแผ่นเดียว เริ่มต้นยุคคอมพิวเตอร์ส่วนบุคคล
  - **ภาพประกอบ:** รูปชิปไมโครชิปสี่เหลี่ยมเรียบง่าย มีขา 4 ด้าน
- **[1 ดาว / 1 แต้ม] Pipelining (ระบบไพป์ไลน์)**
  - **คำอธิบาย UI:** เทคนิคการรีดประสิทธิภาพขั้นสุด โดยจับการประมวลผลมาทำทับซ้อนกันเหมือนสายพานโรงงาน ทำให้ทำงานหลายคำสั่งต่อเนื่องไม่ยอมหยุดพัก
  - **ภาพประกอบ:** เส้นประแบบลูกศรขนานกัน 3 เส้น ทอดยาวไปข้างหน้า

### Tier 2 (6 ใบ)
- **[1 ดาว / 1 แต้ม] Task Scheduler (ตารางจัดการงาน)**
  - **คำอธิบาย UI:** ตัวจัดการเวลาของซีพียู คอยสลับให้แต่ละแอปพลิเคชันได้ใช้พลังประมวลผลอย่างยุติธรรม ไม่มีใครโดนแย่งซีน
  - **ภาพประกอบ:** ไอคอนรูปปฏิทินหรือตารางที่กำลังถูกขีดฆ่างานทิ้ง (Minimal List)
- **[1 ดาว / 1 แต้ม] Interrupt Handler (สับสวิตช์ด่วน)**
  - **คำอธิบาย UI:** กลไกเบรกรถฉุกเฉิน เมื่อมีเหตุการณ์ด่วนจากฮาร์ดแวร์ ซีพียูจะหยุดงานปัจจุบันแล้ววิ่งไปแก้ไขปัญหานั้นทันทีก่อนจะกลับมาทำงานเดิม
  - **ภาพประกอบ:** ไอคอนรูปสายฟ้าฟาดทับเส้นตรง
- **[2 ดาว / 2 แต้ม] Multi-Core (ซีพียูหลายแกน)**
  - **คำอธิบาย UI:** จับเอาซีพียูหลายตัวมายัดรวมในชิปเดียว ช่วยรันหลายแอปพลิเคชันพร้อมกันได้จริงๆ โดยไม่ต้องสลับไปมาอย่างรวดเร็ว
  - **ภาพประกอบ:** สี่เหลี่ยม 4 ช่องที่ประกอบกันเป็นชิป 1 ตัว
- **[2 ดาว / 2 แต้ม] Multithreading (มัลติเธรดดิง)**
  - **คำอธิบาย UI:** พลังของซอฟต์แวร์ในการแตกโปรแกรมยักษ์ออกเป็นสายงานเล็กๆ (Thread) เพื่อให้ซีพียูทำงานขนานกัน ลดอาการแอปค้าง
  - **ภาพประกอบ:** เส้นใย 3 เส้นพันเกลียวหรือพุ่งคู่กัน
- **[3 ดาว / 4 แต้ม] Hyper-Threading (ไฮเปอร์เธรด)**
  - **คำอธิบาย UI:** เทคนิคจำลองแกนซีพียู 1 คอร์ให้เป็น 2 คอร์เสมือน ทำให้ตัวซีพียูรีดเค้นพลังความว่างงานออกมาทำงานได้อีกขั้นโดยไม่เสียเวลา
  - **ภาพประกอบ:** เส้นใยที่แยกออกจากกล่อง 1 กล่องเป็น 2 เส้น
- **[3 ดาว / 4 แต้ม] GPU (หน่วยจัดการกราฟิก)**
  - **คำอธิบาย UI:** ซีพียูกล้ามโตที่อาจไม่ฉลาดรอบด้าน แต่คำนวณเลขเมทริกซ์และประมวลผลพันคำสั่งพร้อมกันได้เร็วกว่าซีพียูปกติเป็นร้อยเท่า!
  - **ภาพประกอบ:** ไอคอนพัดลมระบายความร้อนการ์ดจอแบบแบนราบ

### Tier 3 (4 ใบ)
- **[3 ดาว / 4 แต้ม] Server Farm CPU (คอร์ฟาร์มแม่ข่าย)**
  - **คำอธิบาย UI:** หน่วยประมวลผลเกรดเอ็นเตอร์ไพรส์ สร้างมาเพื่ออึดและรับโหลดจากผู้ใช้งานเป็นแสนคนพร้อมกันได้โดยไม่หลอมละลาย
  - **ภาพประกอบ:** สัญลักษณ์ตู้เซิร์ฟเวอร์เรียงซ้อนกัน
- **[4 ดาว / 5 แต้ม] Neural Processing Unit (NPU)**
  - **คำอธิบาย UI:** ชิปแห่งโครงข่ายประสาทเทียม มีสถาปัตยกรรมเฉพาะไว้บูสต์ความเร็วโลกแห่ง AI การแยกแยะภาพ และงานเรียนรู้เชิงลึกของเครื่องจักร
  - **ภาพประกอบ:** โครงข่ายเส้นเชื่อมจุด (Node & Edge) คล้ายคลื่นสมองมนุษย์
- **[4 ดาว / 5 แต้ม] Quantum Processor / QPU**
  - **คำอธิบาย UI:** ก้าวล่วงขอบเขตฟิสิกส์ดั้งเดิมด้วยควอนตัมที่ข้ามสถานะ 0 กับ 1 แก้ปัญหาสมการที่ซีพียูปกติรันล้านปีให้จบในเสี้ยววินาที
  - **ภาพประกอบ:** จุดศูนย์กลางแวดล้อมด้วยเส้นวงโคจรอิเล็กตรอน
- **[5 ดาว / 7 แต้ม] Supercomputer Cluster (คลัสเตอร์ระดับจักรวาล)**
  - **คำอธิบาย UI:** มหาคอมพิวเตอร์ที่ต่อเครื่องเซิร์ฟเวอร์เรือนแสนตัวเป็นสมองเดียว เพื่อจำลองสภาพอากาศโลก หรือวิเคราะห์โครโมโซมลึกสุดของมนุษย์
  - **ภาพประกอบ:** โครงสร้างจุดที่โยงใยรวมกันเกิดเป็นดวงดาวสว่างตรงกลาง

## 🟦 หมวดหน่วยความจำ (RAM - สีฟ้า)
*โฟกัส: การจดจำชั่วคราว ความเร็วในการเข้าถึง โครงสร้างข้อมูล*

### Tier 1 (8 ใบ)
- **[0 ดาว / 0 แต้ม] Bit (บิตข้อมูล)**
  - **คำอธิบาย UI:** ฝุ่นละอองที่เล็กที่สุดของจักรวาลดิจิทัล แสดงความหมายได้เพียง ใช่ หรือ ไม่ใช่, 0 หรือ 1 เท่านั้น
  - **ภาพประกอบ:** กล่องสี่เหลี่ยมเล็กๆ 1 ใบที่มีเลข 1 อยู่ด้านใน
- **[0 ดาว / 0 แต้ม] Variable (ระบบตัวแปร)**
  - **คำอธิบาย UI:** ลิ้นชักเล็กๆ ใน RAM ที่โค้ดสร้างมาเพื่อเก็บกระดาษโน้ตตัวเลขหรือข้อความชั่วคราวระหว่างการรันโปรแกรม
  - **ภาพประกอบ:** ไอคอนหน้าตัดลิ้นชักเรียบๆ กล่องเดียว
- **[0 ดาว / 0 แต้ม] Pointer (เข็มทิศชี้ตำแหน่ง)**
  - **คำอธิบาย UI:** ตัวแปรสุดพิเศษที่ไม่ได้เก็บว่าค่าคืออะไร แต่เก็บ "ที่อยู่ (Address)" เพื่อชี้วัดว่าของสิ่งนั้นซ่อนอยู่ที่ไหนในความจำ
  - **ภาพประกอบ:** รูปลูกศรชี้ลงไปที่จุดพิกัดวงกลม
- **[0 ดาว / 0 แต้ม] CPU Registers (เรจิสเตอร์)**
  - **คำอธิบาย UI:** หน่วยความจำระดับ VVIP ที่นั่งเก้าอี้ติดซีพียู จุได้น้อยแสนน้อยแต่วิ่งเร็วและราคาแพงที่สุดในระดับฮาร์ดแวร์
  - **ภาพประกอบ:** กล่องเรียงลำดับซ้อนกัน 3 กล่องมีไฟกระพริบ
- **[0 ดาว / 0 แต้ม] Static RAM / SRAM**
  - **คำอธิบาย UI:** แรมสายสปีด ไม่ต้องคอยชาร์จแบตก็จำได้ แต่เพราะมันกินพื้นที่ในการผลิต จึงมักทำเป็นพื้นที่พักบัสข้อมูลชั่วคราว
  - **ภาพประกอบ:** แผ่นวงจรรูปสี่เหลี่ยมผืนผ้าแนวนอน
- **[0 ดาว / 0 แต้ม] Dynamic RAM / DRAM**
  - **คำอธิบาย UI:** แรมคลาสสิกที่ราคาถูกและยัดพื้นที่ได้มหาศาล แต่แลกมาด้วยความจุกจิกที่ต้องเติมไฟฟ้าอยู่ตลอดไม่งั้นลืมหมด
  - **ภาพประกอบ:** ภาพแรมแท่งๆ แบบมินิมอล มีจุด 4 จุดบนแท่ง
- **[1 ดาว / 1 แต้ม] Data Structure Basics (โครงสร้างบิลด์ข้อมูล)**
  - **คำอธิบาย UI:** การสับข้อมูลและจัดเรียงมันลง RAM อย่างมีระเบียบ (เช่น อาเรย์เรียงแถว, หรือสแต็กซ้อนทับ) หาเจอง่ายรันได้คล่อง
  - **ภาพประกอบ:** ช่องบล็อกสามช่องเรียงติดกัน
- **[1 ดาว / 1 แต้ม] Cache Memory L1 (แคชรอยหยักสมอง)**
  - **คำอธิบาย UI:** หลุมหลบภัยข้อมูลจิ๋วที่ฝังในซีพียูคอร์ ป้องกันไม่ให้แอปต้องเสียเวลาเดินออกไปควานหาของจาก RAM ภายนอก
  - **ภาพประกอบ:** ไอคอนชิปซีพียูมีวงกลมเล็กๆ เรืองแสงด้านซ้ายล่าง

### Tier 2 (6 ใบ)
- **[1 ดาว / 1 แต้ม] Memory Allocation (จองพื้นที่ล่วงหน้า)**
  - **คำอธิบาย UI:** การตีอาณาเขตในหน่วยความจำ RAM อย่างชาญฉลาด ร้องขอปุ๊บได้ปั๊บ เพื่อขยายแอปพลิเคชันตามการใช้งาน
  - **ภาพประกอบ:** ตารางตารางกริดที่ถูกไฮไลต์เพียง 4 ช่องตรงกลาง
- **[1 ดาว / 1 แต้ม] Garbage Collection (เทศบาลเก็บขยะความจำ)**
  - **คำอธิบาย UI:** ระบบทำความสะอาดพื้นที่ RAM แบบอัตโนมัติของตัวภาษาโปรแกรม มันจะคอยล้างโค้ดที่แอปฯ ทิ้งร้างเพื่อกันเมมโมรี่เต็ม
  - **ภาพประกอบ:** ไอคอนบล็อกข้อมูลกำลังตกลงในถัง
- **[2 ดาว / 2 แต้ม] Virtual Memory (ยืมพื้นที่มายา)**
  - **คำอธิบาย UI:** เมื่อแรมหมด? ไม่เป็นไร! หยิบยืมพื้นที่จากฮาร์ดดิสก์มาหลอกซีพียูว่ามีแรมเหลือเฟือ แม้จะช้าหน่อยแต่แอปก็ไม่เด้งหลุด
  - **ภาพประกอบ:** เมฆก้อนหนึ่งเชื่อมกับไอคอนชิป RAM
- **[2 ดาว / 2 แต้ม] Paging & Swapping (สลับหน้าเอกสาร)**
  - **คำอธิบาย UI:** อัลกอริทึมจัดการ Virtual Memory ด้วยการแบ่งเนื้อหาเป็นหน้าเอกสาร ดึงแค่หน้าที่ควรอ่านมาเข้า RAM والباقيเก็บไปก่อน
  - **ภาพประกอบ:** รูปกระดาษ 2 แผ่นสลับสับเปลี่ยนตำแหน่งกัน
- **[3 ดาว / 4 แต้ม] L2/L3 Cache Architecture (สถาปัตยกรรมแคชรวมศูนย์)**
  - **คำอธิบาย UI:** แคชส่วนกลางระดับบิ๊กที่เชื่อมคอร์ทุกคอร์ในซีพียูให้แชร์ข้อมูลถึงกันได้ ทำให้ซีพียูไหลลื่นไม่ต้องรอคิวแรมเลยสักนิด
  - **ภาพประกอบ:** ถังหน่วยความจำใหญ่ตรงกลางจ่ายข้อมูลออกไปยังชิปเล็กๆ
- **[3 ดาว / 4 แต้ม] Buffer Overflow Protection (ระบบกั้นเขื่อนข้อมูล)**
  - **คำอธิบาย UI:** ด่านเหล็กที่ปกป้องไม่ให้แฮ็กเกอร์ยัดฟังก์ชันร้ายมาเขียนทับความจำอื่นในแรม ป้องกันระบบรวนกระทันหัน
  - **ภาพประกอบ:** รูปโล่ป้องกันทับอยู่บนกล่องข้อความที่ล้นขอบ

### Tier 3 (4 ใบ)
- **[3 ดาว / 4 แต้ม] Shared Memory Model (พื้นที่สนทนาระหว่างแอป)**
  - **คำอธิบาย UI:** พื้นที่ศูนย์กลางเอกราชบนแรม ที่อนุญาตให้สองโปรแกรมคนละค่ายแวะมาวางเอกสารแบ่งปันกันได้ทันทีแบบไม่ต้องคุยผ่านเครือข่าย
  - **ภาพประกอบ:** ไอคอนแอป 2 อันมีเส้นโค้งชี้เข้าหาพื้นที่ตรงกลางเดียวกัน
- **[4 ดาว / 5 แต้ม] In-Memory Database (ฐานข้อมูลเหินเวหา)**
  - **คำอธิบาย UI:** ยกฐานข้อมูลมาไว้บน RAM ล้วนๆ! ทุกตาราง ทุกเรคเคอร์ดค้นหาเจอในระดับมิลลิวินาที (อย่างเกมที่ตอบสนองไวๆ)
  - **ภาพประกอบ:** รูปถังข้อมูล (Database cylinder) วางอยู่บนชิป RAM
- **[4 ดาว / 5 แต้ม] Distributed Cache (เครือข่ายแรมกระจายศูนย์)**
  - **คำอธิบาย UI:** เชื่อม RAM จากเซิร์ฟเวอร์นับสิบเครื่องให้ทำงานเหมือนRAMบ่อเดียว เพื่อบูสต์การคลิกเข้าเว็บระดับผู้ใช้หลักล้าน
  - **ภาพประกอบ:** เครือข่ายชิป RAM 3 ตัวที่เชื่อมกันเป็นวงกลม
- **[5 ดาว / 7 แต้ม] Holographic Memory (หน่วยความจำ 5 มิติ)**
  - **คำอธิบาย UI:** ปฏิวัติทฤษฎีสสารด้วยการยัดข้อมูลลงในชั้นผลึกสามมิติของเลเซอร์ความเที่ยงตรงสูง จุคลังสมุดภาพลงในขนาดเท่าลูกเต๋า
  - **ภาพประกอบ:** สัญลักษณ์โฮโลแกรมแท่งปริซึมที่แผ่แสงรอบๆ

## ⬜ หมวดพื้นที่เก็บข้อมูล (Storage - สีขาว)
*โฟกัส: ฝากตาราง ฝากไฟล์ ฐานข้อมูลที่ทนทาน*

### Tier 1 (8 ใบ)
- **[0 ดาว / 0 แต้ม] Text File (ไฟล์ตัวอักษรดิบ)**
  - **คำอธิบาย UI:** รูปแบบการบันทึกข้อมูลที่เบสิกที่สุด ใช้ตัวอักขระล้วนๆ คอมพิวเตอร์หรือคนก็เปิดอ่านได้ง่าย เก็บถาวรได้
  - **ภาพประกอบ:** ไอคอนแผ่นกระดาษโดดๆ มีขีดเส้น 2-3 บรรทัด
- **[0 ดาว / 0 แต้ม] Directory (โฟลเดอร์/แฟ้มบัญชี)**
  - **คำอธิบาย UI:** กล่องจำลองที่ระบบปฏิบัติการสร้างเพื่อช่วยแบ่งหมวดหมู่ของไฟล์ ทำให้ดิสก์ที่มีเป็นล้านฟอร์แมตไม่อีรุงตุงนัง
  - **ภาพประกอบ:** ไอคอนรูปแฟ้มเอกสารที่เรียบแบนมากๆ
- **[0 ดาว / 0 แต้ม] Basic Query (สืบค้นและดึงทวีป)**
  - **คำอธิบาย UI:** ประโยคคำสั่งเริ่มต้นที่เราใช้ตะโกนสั่งโปรแกรมเพื่อให้หยิบหนังสือที่ต้องการออกมาจากระบบจัดเก็บ
  - **ภาพประกอบ:** แว่นขยายที่มีเส้นโค้ดหรือฐานข้อมูลอยู่ข้างใน
- **[0 ดาว / 0 แต้ม] Magnetic Hard Disk / HDD**
  - **คำอธิบาย UI:** ของขลังประจำคอมพิวเตอร์ อาศัยแขนหัวกระดาษหมุนจานแม่เหล็กติ๊กๆ อาจจะช้าแต่มันให้พื้นที่เป็นเทราไบต์แบบคุ้มเหลือเชื่อ
  - **ภาพประกอบ:** แผ่นดิสก์วงกลมที่มีเข็มเซนเซอร์พาดทับตรงกลาง
- **[0 ดาว / 0 แต้ม] Solid State Drive / SSD**
  - **คำอธิบาย UI:** ถอดชิ้นส่วนกลไกทิ้ง เปลี่ยนเป็นเซลล์แฟลชล้วนๆ เปลี่ยนโลกการเปิดเครื่องและโหลดเกมที่เคยรอ 1 นาทีเหลือ 5 วินาที
  - **ภาพประกอบ:** แผ่นสี่เหลี่ยม SSD มินิมอลพร้อมลายปริ้นวงจรเล็กน้อย
- **[0 ดาว / 0 แต้ม] FAT Formatter (กระดานแบ่งหน้าเว็บแรก)**
  - **คำอธิบาย UI:** รูปแบบการเตรียมหน้ากระดานบันทึกในดิสก์แบบบุกเบิกสุดๆ ใช้งานง่ายและทุกระบบทั่วโลกยังคงพูดภาษานี้ได้
  - **ภาพประกอบ:** แผ่นดิสก์ฟลอปปี้ 3.5 นิ้วรุ่นเก๋า
- **[1 ดาว / 1 แต้ม] Relational Data Base (ความสัมพันธ์ของกล่องข้อมูล)**
  - **คำอธิบาย UI:** เริ่มต่อจุดระหว่างแผ่นงานต่างๆ เพื่อป้องกันข้อมูลซ้ำซ้อน จัดระบบให้ชัดเจนด้วยกฎเกณฑ์คณิตศาสตร์ (อารมณ์ตาราง Excel ขั้นสูง)
  - **ภาพประกอบ:** ตาราง 2 ตารางที่มีเส้นโยงลูกศรเชื่อมคอลัมน์ถึงกัน
- **[1 ดาว / 1 แต้ม] B-Tree Indexing (รากฐานดัชนีบี-ทรี)**
  - **คำอธิบาย UI:** กลเม็ดสารบัญหลังหนังสือ ที่จำแนกต้นไม้ควานหาตัวอักษรด้วยความไวสูง ทำให้หาผู้ใช้งานคนที่ 1 ล้านเจอในพริบตา
  - **ภาพประกอบ:** แผนผังเส้นต้นไม้คว่ำฐานแคบ (1 กิ่งแตก 2->4)

### Tier 2 (6 ใบ)
- **[1 ดาว / 1 แต้ม] JSON Formatting (พัสดุห่อรหัสซอร์ส)**
  - **คำอธิบาย UI:** โครงสร้างข้อมูลที่เกิดเพื่อชาวเว็บเบราว์เซอร์ ใช้ส่งค่าตัวแปรข้ามอินเตอร์เน็ตในรูปแบบที่เป็นมิตรกับมนุษย์สุดๆ
  - **ภาพประกอบ:** ไอคอนตัวปีกกาที่ข้างในมีเครื่องหมายโคลอน `{ : }`
- **[1 ดาว / 1 แต้ม] Data Compression (ตัวรีดไขมันขยะ)**
  - **คำอธิบาย UI:** อัลกอริทึมน่าทึ่งในการตัดรูปแบบข้อความที่ซ้ำๆ หดย่อขนาดของไฟล์ให้เล็กลงครึ่งหนึ่งก่อนเซฟเก็บ (อย่างพวกซิปไฟล์)
  - **ภาพประกอบ:** สัญลักษณ์ตัวลบรูดซิปลง (ลูกศรบีบอัดของเข้าหากัน)
- **[2 ดาว / 2 แต้ม] SQL Standards (ศูนย์รวมผู้พิทักษ์ตาราง)**
  - **คำอธิบาย UI:** เสาหลักวงการธนาคาร ฐานข้อมูลที่ยึดกฎความปลอดภัย กฎทุกการเงินต้องไม่ตกหล่นแม้มีคนโอนเงินพร้อมกันหมื่นคน
  - **ภาพประกอบ:** ถัง Database ทรงกระบอกที่มีรูปกุญแจหรือตราประทับ
- **[2 ดาว / 2 แต้ม] NoSQL Database (กบฏแหกกฎตาราง)**
  - **คำอธิบาย UI:** เบื่อตาราง? สาดข้อมูลเก็บเป็นเอกสารได้เลย! ถนัดรับข้อมูลล้นทะลักบนแอปมือถือที่รวดเร็ว ยืดหยุ่น ไม่เอาโครงสร้างจุกจิก
  - **ภาพประกอบ:** ถัง Database ที่ถูกทลายกำแพงออกมาเป็นรูปโพลิกอน
- **[3 ดาว / 4 แต้ม] RAID Array (ดิสก์โต้คลื่นสลาตัน)**
  - **คำอธิบาย UI:** จับสหายฮาร์ดดิสก์ 2-3 ตัวมายัดรวมกัน ให้แบ็คอัพก๊อปปี้กันเองแบบสดๆ ถ้าตัวแรกตาย ตัวที่สองรับช่วงต่อทันที ข้อมูลไม่หายไปไหน
  - **ภาพประกอบ:** ก้อนกระบอกตั้งเรียงกัน 3 ก้อนเหมือนแบตเตอรี่ฉนวน
- **[3 ดาว / 4 แต้ม] Snapshot Recovery (เครื่องย้อนเวลากู้ชีพ)**
  - **คำอธิบาย UI:** เทคโนโลยีแคปเจอร์สภาพฮาร์ดดิสก์เอาไว้ทุกระยะ เมื่อฐานข้อมูลล่ม สามารถกระชากสายหมุนเวลากลับไปเมื่อ 5 นาทีก่อนได้ทันที
  - **ภาพประกอบ:** นาฬิกาทวนเข็มซ้อนทับภาพไดสก์

### Tier 3 (4 ใบ)
- **[3 ดาว / 4 แต้ม] Data Warehouse (เหมืองคลังอาวุธองค์กร)**
  - **คำอธิบาย UI:** โกดังยักษ์ที่ดูดข้อมูลย้อนหลัง 10 ปีของบริษัทมาแช่แข็ง ไว้ให้ทีมผู้บริหารใช้วิเคราะห์ประมวลวิสัยทัศน์ในอนาคตอันยาวนาน
  - **ภาพประกอบ:** รูปทรงอาคารโรงเรียน/โกดังอักษร DB ขนาดใหญ่
- **[4 ดาว / 5 แต้ม] Distributed File System (ระบบแฟ้มโปรยเมฆฝน)**
  - **คำอธิบาย UI:** ฉีกทฤษฎีกระจายไฟล์ 1 ไฟล์ลงชิ้นเล็กๆ แอบเก็บเอาไว้ในมือถือและเซิร์ฟเวอร์นับพันเครื่อง (ตัวอย่างวงการบิททอร์เรนต์ หรือ Hadoop)
  - **ภาพประกอบ:** เมฆที่เชื่อมเป็นโครงข่ายหยดน้ำ 5 แฉก
- **[4 ดาว / 5 แต้ม] Cloud Object Storage (อ่างเก็บเนื้อหาคลาวด์)**
  - **คำอธิบาย UI:** เก็บรูปภาพผู้ใช้ระดับพันล้านบัญชีเข้าห้วงนภากาศคลาวด์ ไม่ต้องมาคอยซื้อดิสก์เอง ทนทานหายห่วง สเกลระดับโลกในข้ามคืน (เช่น Amazon S3)
  - **ภาพประกอบ:** รูปเมฆก้อนหนึ่งกำลังโอบล้อมกล่องกระดาษแสนเบา
- **[5 ดาว / 7 แต้ม] Big Data Data Center (ศูนย์กลางสิริมหาสมุทรบิ๊กดาต้า)**
  - **คำอธิบาย UI:** ยุคสมัยของ Data is the new oil. โครงสร้างฮาร์ดแวร์มโหฬารเพื่อรองรับการเจาะพฤติกรรมมนุษย์ ทำนายเทรนด์และล้างไพ่บริษัทคู่แข่ง
  - **ภาพประกอบ:** สัญลักษณ์ศูนย์ข้อมูลที่มีคลื่นสัญญาณวิทยุกระจายแสงแผ่ออกไปกว้างจนสุดขอบ

## ⬛ หมวดเครือข่ายอินเทอร์เน็ต (Network - สีดำ/เทาเข้ม)
*โฟกัส: เส้นทางจราจร โปรโตคอล สะพาน และเมฆ*

### Tier 1 (8 ใบ)
- **[0 ดาว / 0 แต้ม] LAN Cable (สายสื่อสารเส้นเลือด)**
  - **คำอธิบาย UI:** สายทองแดงเกลียวที่นำพาโลกคอมสองเครื่องมาบรรจบคุยกันได้ การเริ่มต้นของการสร้างอินทราเน็ตสำนักงานที่มั่นคงสุดๆ
  - **ภาพประกอบ:** ไอคอนหัวต่อสาย LAN (RJ45) หันหน้าตรงแบบแบน
- **[0 ดาว / 0 แต้ม] MAC Address (ทะเบียนรถดิจิทัล)**
  - **คำอธิบาย UI:** หมายเลขเครื่องที่ไม่ซ้ำใครในใต้หล้า ฝังติดตัวอุปกรณ์มาจากโรงงานผู้ผลิต เปรียบเสมือนบัตรประชาชนของฮาร์ดแวร์ก่อนจะออกสู่อินเทอร์เน็ต
  - **ภาพประกอบ:** ป้ายบาร์โค้ดขนาดจิ๋วที่มีเลขฐานสิบหก 0A-B3...
- **[0 ดาว / 0 แต้ม] Network Hub (โง่แต่ขยันฮับโหนด)**
  - **คำอธิบาย UI:** อุปกรณ์สับสวิตช์ยุคบุกเบิกที่เชื่อม 4 คอมพิวเตอร์เข้าด้วยกัน ข้อเสียคือใครส่งมา มันจะตะโกนฟ้องให้คอมทุกเครื่องได้ยินพร้อมกัน(รถติดง่าย)
  - **ภาพประกอบ:** สี่เหลี่ยมแบนๆ ที่พอร์ตต่อกระจายออก 4 รู
- **[0 ดาว / 0 แต้ม] IP Address (พิกัดบ้านเลขที่อินเทอร์เน็ต)**
  - **คำอธิบาย UI:** บ้านเลขที่เป็นตัวเลข (เช่น 192.168...) ให้บุรุษไปรษณีย์รู้ว่าคอมเครื่องไหนกำลังเรียกดูเว็บและจะเอาข้อมลกลับมาส่งที่ไหน
  - **ภาพประกอบ:** ป้ายหมุดพิกัด Location map ตรงกลางมีโลกอินเทอร์เน็ต
- **[0 ดาว / 0 แต้ม] Binary Packet (พัสดุย่อยบรรจุกล่องเลข)**
  - **คำอธิบาย UI:** หน้าเว็บไม่เคยส่งมาทั้งก้อน! กฎเครือข่ายต้องหั่นไฟล์ออกเป็นจิ๊กซอว์ล้านชิ้นแล้วแปะบ้านเลขที่โยนเข้าสายเน็ตเพื่อกันสายอุดตัน
  - **ภาพประกอบ:** กล่องพัสดุกระดาษที่มีลายสัญญาณเน็ต
- **[0 ดาว / 0 แต้ม] Modem (สัญญาณตอสัญญาณผี)**
  - **คำอธิบาย UI:** จากยุคสายโทรศัพท์ที่ต๊าดๆๆ ตัวแปลสัญญาณแอนะล็อกของสายไฟธรรมดาเข้าสู่เสียงดิจิทัล สะพานสู่นอกบ้าน
  - **ภาพประกอบ:** กล่องเร้าเตอร์ทรงสูงปล่อยรัศมีคลื่นขึ้นไป 3 ระดับ
- **[1 ดาว / 1 แต้ม] Switch (ตำรวจจราจรอัจฉริยะ)**
  - **คำอธิบาย UI:** ฮับเวอร์ชันฉลาดขึ้นมาหน่อย ที่ฉลาดจำทะเบียนบ้านเครื่องลูกข่าย ส่งพัสดุพุ่งตรงไปเฉพาะผู้รับที่ถูกต้อง รถไม่ติดบนสาย LAN
  - **ภาพประกอบ:** ไอคอนกล่องสี่เหลี่ยมที่มีลูกศรชี้วิ่งไขว้กันตรงกลาง
- **[1 ดาว / 1 แต้ม] HTTP Protocol (ภาษาคนคุยกับเว็บ)**
  - **คำอธิบาย UI:** ภาษากลางโลกของเบราว์เซอร์ ที่อนุญาตให้คุณคลิกลิงก์และดูหน้าเพจเว็บไซต์ธรรมดาๆ กฎพื้นฐานของการเล่นเน็ตแห่งมวลมนุษยชาติ
  - **ภาพประกอบ:** ตัวหนังสือทรงโค้ง HTTP อยู่บนเวิล์ดไวด์เว็บ

### Tier 2 (6 ใบ)
- **[1 ดาว / 1 แต้ม] DHCP Server (เสมียนจ่ายป้ายทะเบียน)**
  - **คำอธิบาย UI:** ซับระบบที่คอยขานเลขบ้าน IP แจกให้เครื่องมือถือหรือคอมใหม่ๆ ที่แหย่หน้าเข้ามาร่วมต่อเน็ตอย่างอัตโนมัติไม่ต้องกรอกเอง
  - **ภาพประกอบ:** แผ่นป้ายที่ถูกหมุนออกจากเครื่องพิมพ์ป้ายทะเบียน
- **[1 ดาว / 1 แต้ม] DNS Registry (สมุดหน้าเหลืองอินเทอร์เน็ต)**
  - **คำอธิบาย UI:** คุณจำเลข IP 142.250.70 ไม่ได้หรอก! DNS ช่วยแปลชื่อ "google.com" ให้กลายเป็นตัวเลขเนิร์ดๆ เอง ช่วยมนุษย์ประหยัดความจำเยอะ
  - **ภาพประกอบ:** สมุดโน้ตปกแข็งที่มีไอคอนโลกประทับ
- **[2 ดาว / 2 แต้ม] Router (บุรุษไปรษณีย์ข้ามพรมแดน)**
  - **คำอธิบาย UI:** ไม่ได้เก่งแค่ในองค์กร เร้าเตอร์คือเข็มทิศพาสายพัสดุของคุณวิ่งลอดอุโมงค์ปลาฉลามใต้มหาสมุทรออกไปยังเซิร์ฟเวอร์ประเทศอื่น
  - **ภาพประกอบ:** วงกลมลูกโลกที่มีลูกศรวิ่งวนแผ่ออกเป็นเส้นใย
- **[2 ดาว / 2 แต้ม] TCP / IP Protocol (สัญญาตราประทับร้อยเปอร์เซ็นต์)**
  - **คำอธิบาย UI:** กฎจราจรขั้นเด็ดขาดที่บังคับว่า "จะเล่นเน็ต พัสดุต้องถึงมือ 100%" หากตกหล่นระหว่างทางTCPจะสั่งให้ส่งใหม่ซ้ำๆ จนกว่าจะครบบริบูรณ์
  - **ภาพประกอบ:** ตราประทับแสตมป์ยาง (Approved) บนกล่องแพกเกต
- **[3 ดาว / 4 แต้ม] Load Balancer (โคนันจัดการฝูงชน)**
  - **คำอธิบาย UI:** ตัวจับกระจายทราฟฟิก พอมีคนกดจองตั๋วศิลปินล้านคนพร้อมกัน เจ้านี่จะรีบหรี่คนเข้ากระจายไปเซิร์ฟเวอร์อื่นๆ ให้เท่ากัน ไม่ให้เครื่องไหนควันขึ้น
  - **ภาพประกอบ:** ตาชั่งสมดุลที่สองฝั่งรองรับเซิร์ฟเวอร์ก้อนใหญ่
- **[3 ดาว / 4 แต้ม] Web Socket (ท่อสายตรงสองฝั่ง)**
  - **คำอธิบาย UI:** อัพเกรดการคุยเว็บจากที่ต้องคอยรีเฟรช เปลี่ยนเป็นการเปิดท่อคอลทางไกลทิ้งไว้ ทำให้โลกเกิดแอพพวก Live Chat และเล่นเกมออนไลน์ขึ้น!
  - **ภาพประกอบ:** ท่อแก้วใส 2 ท่อที่เปิดมีแสงเลเซอร์พุ่งสวนกัน

### Tier 3 (4 ใบ)
- **[3 ดาว / 4 แต้ม] Content Delivery Network / CDN (นินจาส่งของระยะประชิด)**
  - **คำอธิบาย UI:** เซิร์ฟเวอร์ต้นทางอยู่เมกา แต่ก๊อปปี้คลิปวิดีโอไปซ่อนตามทุกทวีป เมื่อคนไทยกดดูไม่ต้องโหลดข้ามโลก แต่ดึงไฟล์จากประเทศสิงคโปร์แทน!
  - **ภาพประกอบ:** ลูกโลกล้อมรอบด้วยจุดโหนดวงกลมย่อย 5 วง
- **[4 ดาว / 5 แต้ม] Cloud Mesh Network (ดาวเทียมใยแมงมุม)**
  - **คำอธิบาย UI:** เครือข่ายการร้อยเรียงเซิร์ฟเวอร์ย่อยเข้ากันคล้ายตาข่าย ตัดชิ้นไหนขาด โหนดอื่นก็รักษาการเชื่อมต่อทดแทนกันได้ไม่มีหยุดพักทำงาน
  - **ภาพประกอบ:** ตาข่ายรูปหกเหลี่ยมรังผึ้งต่อกัน 3 ชิ้น
- **[4 ดาว / 5 แต้ม] Optical Fiber Backbone (วาฬสื่อสารหมื่นกิโลเมตร)**
  - **คำอธิบาย UI:** แกนกระดูกสันหลังอินเทอร์เน็ตที่ยิงคลื่นแสงผ่านท่อใยแก้วขนาดเล็กด้วยความเร็วแสงข้ามแอตแลนติก เป็นรากฐานแบนด์วิดท์มหาศาลยุค 5G
  - **ภาพประกอบ:** สายเคเบิลเรืองแสงเป็นคลื่นเส้นเรียวเล็กผ่านทะเล
- **[5 ดาว / 7 แต้ม] InterPlanetary File System (ระบบแฟ้มกาแล็กซี่ IPFS)**
  - **คำอธิบาย UI:** สถาปัตยกรรมปฏิวัติวงการเว็บให้ทำงานแบบเพียร์-ทู-เพียร์ล้วน ลบศุนย์กลางทิ้ง อินเทอร์เน็ตยุคเว็บ 3.0 ที่ไม่มีใครลบหรือปิดโดเมนเนมลงได้!
  - **ภาพประกอบ:** ดาวเคราะห์ดวงเล็กๆ 3 ดวงร้อยเชือกติดกันบนอวกาศมินิมอล

## 🟨 หมวดความปลอดภัย (Security - สีเหลือง)
*โฟกัส: โล่ป้องกัน ไวรัส กุญแจเข้ารหัส และกำแพงเมือง*

### Tier 1 (8 ใบ)
- **[0 ดาว / 0 แต้ม] Username Password (ประตูรหัสผ่าน)**
  - **คำอธิบาย UI:** คลาสสิก ยั่งยืน และมีปัญหาผู้คนลืมบ่อยที่สุด! รหัสผ่านพื้นฐานกำแพงชั้นแรกของการคัดกรอง "คุณใช่เจ้าของตัวจริงหรือไม่?"
  - **ภาพประกอบ:** กุญแจฟันปลาคลาสสิค 1 ดอกไขอยู่กับตัวหนังสือรูปร่างมนุษย์
- **[0 ดาว / 0 แต้ม] Antivirus (ตัวสเปรย์ฆ่าแมลง)**
  - **คำอธิบาย UI:** โปรแกรมคอยเฝ้าจับตาย ซอฟต์แวร์ยิ้มหวานที่มีพฤติกรรมน่าสงสัย มันจะสแกนไฟล์เพื่อลบล้างไวรัสรุ่นคุณปู่ให้สิ้นซากจากฮาร์ดดิสก์
  - **ภาพประกอบ:** โล่ป้องกันเรียบๆ มีสัญลักษณ์เครื่องหมายบวก (Health) ด้านใน
- **[0 ดาว / 0 แต้ม] Plaintext (หนังสือไม่เข้าเล่มรหัส)**
  - **คำอธิบาย UI:** ความไม่ใส่ใจ การสื่อสารบนเครือข่ายด้วยตัวหนังสือธรรมดาที่ปลิวว่อนผ่านเราเตอร์ ใครต่อ Wi-Fi ผีก็ดักอ่านรหัสผ่านคุณได้ทั้งนั้น
  - **ภาพประกอบ:** รูปกระดาษ 1 แผ่นที่ถูกวาดรูปกะโหลกเตือนภัยแบนราบ (สื่อถึงช่องโหว่)
- **[0 ดาว / 0 แต้ม] Access Token (บัตรผ่านนิติบุคคล)**
  - **คำอธิบาย UI:** เข้าสู่ระบบเสร็จก็จะได้ป้ายโชว์แขวนคอ ไว้ให้เซิร์ฟเวอร์รู้ว่าเห้ยนี่ลูกบ้านล็อคอินเข้าแล้วจะได้ไม่ต้องคอยถามรหัสผ่านทุกครั้งที่คลิกหน้าใหม่
  - **ภาพประกอบ:** ตราประทับรูปตั๋วทองคล้องสายเชือก (Badge id)
- **[0 ดาว / 0 แต้ม] System Logs (กล้องวงจรปิดโค้ด)**
  - **คำอธิบาย UI:** แฟ้มบันทึกเวลาและหลักฐานกรรมที่แอปทุกแอปต้องทิ้งเศษขนมปังไว้ เวลาระบบถูกแฮก ตำรวจไอทีจะดูไฟล์ Log เป็นอันดับแรกว่าใครทำอะไร
  - **ภาพประกอบ:** ม้วนกระดาษที่หลุดออกมาเหมือนใบเสร็จยาวๆ
- **[0 ดาว / 0 แต้ม] Port Blocking (การอุดรูกุญแจประตูเมือง)**
  - **คำอธิบาย UI:** เซิร์ฟเวอร์ 1 เครื่องมีประตู 65,000 บาน การสั่งปิดพอร์ตจราจรที่ไม่ได้ใช้งานคือพื้นฐานความปลอดภัยเบื้องต้นของผู้ดูแลระบบ
  - **ภาพประกอบ:** สัญลักษณ์ช่องกุญแจโดนเส้นสีแดงกากบาททับ
- **[1 ดาว / 1 แต้ม] Salting Password (เหยาะเกลือลงรหัสผ่าน)**
  - **คำอธิบาย UI:** เปลี่ยนอักษร 'admin' ให้กลายเป็นข้อความมั่วๆ ซับซ้อนด้วยการเติมอักขระสุ่มเพิ่มไปท้ายพาสเวิร์ด ป้องกันแฮกเกอร์แกะรหัสย้อนกลับ
  - **ภาพประกอบ:** ขวดเกลือที่เคาะลงบนกล่องข้อความ
- **[1 ดาว / 1 แต้ม] Symmetric Encryption (กุญแจเข้ารหัสสมมาตรดอกเดียว)**
  - **คำอธิบาย UI:** นำข้อมูลมาสับยำแบบคณิตศาสตร์ ล็อคและไขเปิดด้วย "รหัสปุ่มเดียวกัน" เร็วรุนแรง แต่ง่ายต่อการโดนแย่งกุญแจระหว่างทาง
  - **ภาพประกอบ:** ล็อกแม่กุญแจที่มีตัวเลขศูนย์กับหนึ่งเรียงรอบๆ

### Tier 2 (6 ใบ)
- **[1 ดาว / 1 แต้ม] Asymmetric Key / RSA (กุญแจเข้ารหัสแฝดคนละฝา)**
  - **คำอธิบาย UI:** มีกุญแจ 2 ดอก (กุญแจสาธารณะ กับ กุญแจส่วนตัว) แจกดอกแรกให้ชาวโลกส่งของมาเซฟๆ ส่วนตอนเปิดหยิบออกใช้ดอกในห้องนอนไขได้ผู้เดียว!
  - **ภาพประกอบ:** แม่กุญแจที่ถูกแบ่งครึ่งสลับด้าน (สีเข้มครึ่ง อ่อนครึ่ง)
- **[1 ดาว / 1 แต้ม] Firewall Rules (กำแพงไฟจราจร)**
  - **คำอธิบาย UI:** พนักงานตรวจพิสูจน์อักษรพัสดุหน้าเซิร์ฟเวอร์ กำหนดรูลส์เคร่งครัดว่าไอพีดาร์กเว็บไหนห้ามผ่านเข้ามาแพร่เชื้อเด็ดขาด
  - **ภาพประกอบ:** กำแพงอิฐก้อนๆ แบนๆ มีเปลวไฟเล็กๆ บังหน้า
- **[2 ดาว / 2 แต้ม] HTTPS / SSL (ใบรับรองความปลอดภัยเว็บ)**
  - **คำอธิบาย UI:** ติดอาวุธเสริมให้เบราว์เซอร์ด้วยอุโมงค์เขียวแห่งความปลอดภัย ครอบทุกพัสดุเป็นรหัสชั้นพรีเมียม ใครแอบแฮคดูกลางทางจะเห็นแค่ขยะภาษาต่างดาว
  - **ภาพประกอบ:** แม่กุญแจสีเขียวที่ถูกบีบอัดลงไปในกล่องค้นหา Google/URL
- **[2 ดาว / 2 แต้ม] DDoS Mitigation (โล่สะท้อนม็อบซอมบี้)**
  - **คำอธิบาย UI:** เมื่อแฮกเกอร์ระดมกองทัพซอมบี้พันล้านเครื่องมารุมคลิกเว็บคุณ โล่นี้จะกรองแต่ฝูงซอมบี้ให้ชิ่งกลับไป และปล่อยให้ลูกค้าตัวจริงเข้ามาใช้เว็บได้ตามปกติ
  - **ภาพประกอบ:** สัญลักษณ์รังสีคลื่น 3 แลกพุ่งมากระทบหน้าปัดโล่คางหมู
- **[3 ดาว / 4 แต้ม] Two-Factor Auth / 2FA (ด่านกักกันระบบสองชั้น)**
  - **คำอธิบาย UI:** ได้รหัสผ่านหลุดไปแล้วไง? ต่อให้ฟ้าถล่มดินทลาย คนร้ายก็ยังต้องการเลข OTP จากมือถือในกระเป๋ากางเกงของคุณถึงจะแฮกเข้าแอคเคาท์ได้อยู่ดี
  - **ภาพประกอบ:** สมาร์ตโฟนแสดงสายฟ้าฟาดมีลูกกุญแจขึ้นมาหน้าจอ
- **[3 ดาว / 4 แต้ม] Penetration Testing / Pentest (แฮกเกอร์หมวกขาว)**
  - **คำอธิบาย UI:** จ้างนักเจาะระบบมาแฮ็กบริษัทตัวเองก่อนที่โจรจริงๆ จะลงมือ หาช่องโหว่และอุดรอยร้าวทั้งหมด เพื่อการันตีความปลอดภัยก่อนปล่อยซอฟต์แวร์
  - **ภาพประกอบ:** หมวกตีกอล์ฟหมวกปีกแบบนักสืบ (Hacker White Hat) มินิมอล

### Tier 3 (4 ใบ)
- **[3 ดาว / 4 แต้ม] JWT (JSON Web Token) (ใบเบิกทางดิจิทัลพกพา)**
  - **คำอธิบาย UI:** ลาก่อนความจำฝั่งเซิร์ฟเวอร์ ออกตั๋วรับรองมาตรฐานฝังลายเซ็นลงไปให้ผู้ใช้งานพกติดตัว มันสามารถตรวจสอบตัวตนได้รวดเร็วทันใจรองรับสเกลแอปหมื่นคนสบายๆ
  - **ภาพประกอบ:** สัญลักษณ์บัตรตั๋วลอยขึ้นมาถูกประทับตราสีเขียว
- **[4 ดาว / 5 แต้ม] Blockchain Ledger (สมุดขึงบัญชีบล็อกเชน)**
  - **คำอธิบาย UI:** โซ่ตรวนเก็บข้อมูลแบบฝังใจ ไม่สามารถย้อนไปลบหรือแก้ไขอดีตได้เลย เพราะข้อมูลถูกแชร์ให้ผู้ใช้ทุกคนถือสำเนาความจริงกันคนละชุด
  - **ภาพประกอบ:** กล่องลูกบาศก์ 3 ชิ้นเชื่อมต่อด้วยสายโซ่เส้นหนา
- **[4 ดาว / 5 แต้ม] End-to-End Encryption (อุโมงค์เงียบระดับผู้แฝงตัว)**
  - **คำอธิบาย UI:** ปลอดภัยเบอร์แรง แม้แต่เจ้าของแพลตฟอร์มเซิร์ฟเวอร์ หรือรัฐบาลขอเรียกดูข้อมูลแชทก็อ่านไม่ออก! ถอดรหัสได้แค่เครื่องคนรับกับคนส่งเท่านั้น!
  - **ภาพประกอบ:** โทรศัพท์ 2 เครื่องมีสายฟ้า/รหัส 01 วิ่งระหว่างกลางพร้อมกุญแจคล้อง
