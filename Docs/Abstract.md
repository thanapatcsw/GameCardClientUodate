# บทคัดย่อ

โครงงานวิจัยนี้มีวัตถุประสงค์เพื่อออกแบบและพัฒนาแอปพลิเคชันเกมการ์ดเชิงศึกษาบนอุปกรณ์เคลื่อนที่ เพื่อส่งเสริมการเรียนรู้แนวคิดสถาปัตยกรรมระบบสารสนเทศและการบริหารจัดการทรัพยากรคอมพิวเตอร์ โดยประยุกต์ใช้กลไกการบริหารทรัพยากรจากเกมกระดานค้าเพชร (Splendor) ร่วมกับการจำลองกระบวนการประกอบโมดูลซอฟต์แวร์ ผู้เล่นจะได้รับบทบาทเป็นนักพัฒนาระบบรุ่นใหม่ที่ต้องบริหารทรัพยากรระบบคอมพิวเตอร์ 5 ประเภท ได้แก่ CPU, RAM, Network, Storage และ Security เพื่อนำไปพัฒนาการ์ดโมดูลซอฟต์แวร์ที่แบ่งตามระดับความซับซ้อนออกเป็น 3 ระดับ (Tiers 1–3) เพื่อสะสมคะแนนแต้มระบบ (Stars) ให้ครบ 20 แต้มเป็นคนแรกเพื่อชัยชนะ

ระบบโครงสร้างสถาปัตยกรรมได้รับการพัฒนาด้วยเครื่องมือ Unity และภาษา C# โดยรองรับโหมดการเล่นแบบผู้เล่นคนเดียว (Single Player) ที่แข่งขันกับผู้เล่นจำลองคอมพิวเตอร์ (Bot) ซึ่งควบคุมการทำงานด้วยตรรกะเชิงกฎเกณฑ์ (Rule-Based System) ที่มีลำดับการตัดสินใจประเมินกระดาน 4 ระดับ และระบบหน่วงเวลาการตอบคำถามควิซให้ใกล้เคียงพฤติกรรมมนุษย์ สำหรับโหมดการเล่นหลายคนแบบเรียลไทม์ (Real-time Multiplayer) รองรับผู้เล่น 2–4 คน ผ่านเครือข่ายความเร็วสูงด้วย Photon Fusion (Shared Mode) ตัวเกมบูรณาการร่วมกับฐานข้อมูลคลาวด์หลังบ้านของ Supabase และ PostgreSQL ซึ่งจัดการระบบสมัครสมาชิก ล็อกอินผู้ใช้งาน รวมถึงระบบคิวจับคู่อัตโนมัติ (Auto Matchmaking) ผ่านฟังก์ชันระดับฐานข้อมูล (`claim_matchmaking_room`) และจัดเก็บโปรไฟล์ผู้เล่น สถิติ MMR ตลอดจนสกินกรอบรูปโปรไฟล์อวตาร์ของผู้ใช้งานอย่างเป็นรูปธรรม นอกจากนี้ยังมีการบูรณาการระบบแข่งขันตอบคำถามความรู้ด้านคอมพิวเตอร์ (Interventional Quiz) ทุกๆ 3 รอบการเล่นเพื่อจัดลำดับเทิร์น และระบบทดสอบประจำวัน (Daily Quiz) เพื่อสะสมเพชร (Gems) สำหรับนำมาแลกกรอบรูปโปรไฟล์ในระบบร้านค้าในเกม

ผลการดำเนินงานโครงงานพบว่า ระบบซอฟต์แวร์และเกมจำลองการบริหารทรัพยากรระบบสารสนเทศนี้ สามารถทำงานร่วมกันได้อย่างสมบูรณ์และมีความเสถียรตามขอบเขต TOR ตัวเกมทำหน้าที่เป็นสื่อการเรียนรู้เชิงปฏิสัมพันธ์ที่ช่วยแปลงแนวคิดเชิงทฤษฎีที่เป็นนามธรรมให้อยู่ในรูปที่เข้าใจง่าย และเป็นต้นแบบสถาปัตยกรรมการพัฒนาเกมหลายผู้เล่นแบบ Full-Stack ที่เชื่อมต่อระบบคลาวด์ยุคปัจจุบันได้อย่างมีประสิทธิภาพ

- Abstract
  - This research project aims to design and develop an educational mobile card game application to enhance the learning of information technology architecture concepts and computer resource management. By adapting the resource management mechanics of the successful board game "Splendor" and simulating the assembly process of software modules, players assume the role of modern systems developers. They must manage five core computer system resources—namely CPU, RAM, Network, Storage, and Security—to develop software module cards across three levels of complexity (Tiers 1–3), striving to be the first to accumulate 20 system points (Stars) for victory.
  - The system architecture is implemented using the Unity Engine and C#. It supports a Single Player mode competing against a rule-based artificial intelligence player (Bot) featuring a four-level priority decision tree and a randomized quiz response delay simulating human behavior. For the Real-time Multiplayer mode, 2 to 4 players are supported via a high-speed network synchronized using Photon Fusion in Shared Mode. The application is integrated with a cloud-based Supabase backend and PostgreSQL database, managing user registration, authentication, and a real-time matchmaking queue powered by a database-level function (claim_matchmaking_room). Additionally, it stores user profiles, Matchmaking Rating (MMR) statistics, and customizable avatar frame skins. Educational engagement is reinforced by an interventional academic quiz triggered every three game turns to determine the player turn order for the next cycle, alongside a Daily Quiz feature where players can solve computer science challenges to earn Gems for unlocking avatar frames in the in-game cosmetic shop.
  - The project implementation outcomes demonstrate that this resource management simulation game functions seamlessly and stably, fully satisfying all Terms of Reference (TOR) requirements. The application serves as an effective interactive learning medium that translates abstract computer science theories into an intuitive, engaging activity, establishing a robust prototype for modern full-stack multiplayer educational games integrated with state-of-the-art cloud platforms.
