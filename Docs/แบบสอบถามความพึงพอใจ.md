# แบบสอบถามความพึงพอใจ
## การพัฒนาแอปพลิเคชันระบบเกมการ์ดเสริมสร้างแนวคิดเทคโนโลยีสารสนเทศบนเครือข่าย
### (Academic Card Game for Information Technology on Network)

---

**คำชี้แจง:** แบบสอบถามฉบับนี้จัดทำขึ้นเพื่อสำรวจความพึงพอใจของผู้ใช้งานที่มีต่อแอปพลิเคชันระบบเกมการ์ดเสริมสร้างแนวคิดเทคโนโลยีสารสนเทศบนเครือข่าย ข้อมูลที่ได้รับจะนำไปใช้เพื่อการพัฒนาและปรับปรุงระบบให้มีประสิทธิภาพยิ่งขึ้น ผู้ตอบแบบสอบถามไม่จำเป็นต้องระบุชื่อ และข้อมูลทุกอย่างจะถูกเก็บเป็นความลับ

**เกณฑ์การให้คะแนน (Likert Scale 5 ระดับ):**

| ระดับคะแนน | ความหมาย |
|:---:|:---|
| 5 | พึงพอใจมากที่สุด |
| 4 | พึงพอใจมาก |
| 3 | พึงพอใจปานกลาง |
| 2 | พึงพอใจน้อย |
| 1 | พึงพอใจน้อยที่สุด |

---

## ส่วนที่ 2: แบบประเมินความพึงพอใจ

### ด้านที่ 1: การออกแบบและความสะดวกในการใช้งาน (User Interface & User Experience)

| ข้อ | รายการประเมิน | 5 | 4 | 3 | 2 | 1 |
|:---:|:---|:---:|:---:|:---:|:---:|:---:|
| 1 | ความสวยงามของธีมกราฟิกและสัญลักษณ์ระบบสารสนเทศ (CPU, RAM, Network, Storage, Security) ที่ใช้ในเกม | | | | | |
| 2 | ความเหมาะสมของการจัดวางเมนู ปุ่มควบคุม และตำแหน่งแสดงทรัพยากรบนกระดานเกม | | | | | |
| 3 | ความถูกต้องและใช้งานง่ายของระบบร้านค้าแลกเปลี่ยนกรอบรูปโปรไฟล์ (Avatar Shop) | | | | | |
| 4 | ความลื่นไหลของการเปลี่ยนหน้าจอระหว่างฉากต่างๆ และความเหมาะสมของธีมสีในเกม | | | | | |

### ด้านที่ 2: ระบบการเล่นเกมการ์ดหลักและการแลกเปลี่ยนทรัพยากร (Core Board Game & Resource Mechanics)

| ข้อ | รายการประเมิน | 5 | 4 | 3 | 2 | 1 |
|:---:|:---|:---:|:---:|:---:|:---:|:---:|
| 5 | ความเข้าใจง่ายของขั้นตอนการซื้อและจองการ์ดโมดูลคอมพิวเตอร์ในแต่ละเทิร์น | | | | | |
| 6 | ความถูกต้องและความสมดุลของระบบหักลดเหรียญทรัพยากรในการพัฒนาการ์ดแต่ละใบ | | | | | |
| 7 | ความถูกต้องของการบันทึกสถิติแพ้ชนะและระบบสิ้นสุดเกมเมื่อสะสมครบ 20 ดาว | | | | | |

### ด้านที่ 3: ระบบการจับคู่ออนไลน์และการตอบคำถามควิซชิงสิทธิ์เทิร์น (Real-time Matchmaking & Interventional Quiz)

| ข้อ | รายการประเมิน | 5 | 4 | 3 | 2 | 1 |
|:---:|:---|:---:|:---:|:---:|:---:|:---:|
| 8 | ความรวดเร็วและประสิทธิภาพของระบบค้นหาห้องจับคู่ผู้เล่นออนไลน์ | | | | | |
| 9 | ความเหมาะสมของระดับความยาก ความท้าทาย และเนื้อหาของคำถามควิซวิชาคอมพิวเตอร์ | | | | | |
| 10 | ความเหมาะสมของกรอบเวลานับถอยหลัง 10 วินาทีในการชิงสิทธิ์ลำดับเทิร์นการเล่น | | | | | |
